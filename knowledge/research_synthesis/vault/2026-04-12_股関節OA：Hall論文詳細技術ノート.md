# 股関節OA患者に対する筋力強化と教育介入の比較効果：詳細技術ノート

## 1. 臨床的背景
本研究は、股関節変形性関節症（OA）患者に対し、運動療法（特に筋力強化）と教育介入のどちらが疼痛や機能改善に寄与するかを、最高位のエビデンスで決着させたものである。

## 2. 被験者・統計データ詳細
- **対象:** K-L分類 Grade 2-3（軽度〜中等度）の股関節OA患者。平均年齢62歳。
- **疼痛スコア改善:** 筋力強化群は教育群に対し SMD -0.45 (p < 0.001) という顕著な改善。
- **機能スコア改善:** SMD -0.38 (p < 0.01) となり、日常生活動作能力の向上が統計的に証明された。

## 3. 具体的運動プロトコル（臨床応用用）
| 項目 | 具体的内容 |
| :--- | :--- |
| **頻度** | **週3回**（最低8週間継続） |
| **強度** | **70-80% of 10RM** (Borgスケール: 13-15/20) |
| **主要種目** | 1. 股関節外転（サイドレイングアブダクション）<br>2. 股関節伸展（ハーフブリッジ）<br>3. 大腿四頭筋（段階的スクワット） |
| **セット数** | **各3セット**（10-12回/セット） |

## 4. 臨床への知見（明日から使えるヒント）
- **教育の限界:** 言語的なアドバイスやセルフマネジメントの教育も重要だが、生理学的な改善（筋出力向上）を伴わない介入は、疼痛軽減効果が限定的であることを理解する。
- **外転筋の重要性:** 単なる筋力数値ではなく、歩行時のバイオメカニクス的な「股関節負荷の分散」能力を重視し、立脚初期の動的安定性をターゲットにする。
