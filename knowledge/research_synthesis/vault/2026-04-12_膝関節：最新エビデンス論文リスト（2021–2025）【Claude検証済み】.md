### 📄 論文 ① ✅ 検証済み

**タイトル（英）：** Comparative effectiveness of exercise modalities for knee osteoarthritis: a systematic review and network meta-analysis of 217 RCTs  
**タイトル（日）：** 膝OAに対する運動療法モダリティの比較有効性：217本のRCTを用いたネットワークメタ分析

| 項目 | 内容 |
|------|------|
| **著者** | Yan et al. |
| **発行年** | 2025年10月 |
| **掲載誌** | *The BMJ* |
| **DOI** | `10.1136/bmj-2025-085242` |
| **リンク** | https://pubmed.ncbi.nlm.nih.gov/41093618/ |
| **EBMレベル** | ★★★★★ ネットワークメタ分析（最高位） |
| **確認状況** | ✅ PubMed PMID: 41093618 |

**💎 資産価値（1行）：**  
217本・15,684名という圧倒的サンプル数で「有酸素運動が膝OAの第一選択」を中等度確実性で証明した、現時点の到達点論文。

**🔑 最重要ポイント：**
- 有酸素運動（歩行・自転車・水泳）が疼痛・機能・歩行・QOLの全アウトカムで最高効果を示し、第一選択介入として推奨される。
- マインドボディ運動は短期機能に、ニューロモーター運動は短期歩行に、柔軟性運動は長期疼痛に、それぞれ特異的な優位性を持つ。

膝蓋大腿部痛（PFP）の治療選択において「股関節＋膝複合強化」が最も有効であることを明示し、PT介入の優先順位を根拠立てて構築できる。

**🔑 最重要ポイント：**
- 股関節＋膝の複合強化（Hip-and-Knee Exercise）は、膝単独強化より疼痛・機能の両面で有意に優れた結果を示す。
- 患者特性（疼痛破局化・BMI・ベースライン疼痛強度）によって最適な運動種別が異なり、個別化プログラム設計の根拠を提供する。

## 🧠 臨床への統合メモ

### 膝OAのPT戦略
```
有酸素運動（第一選択）← 論文①で証明（217 RCT, n=15,684）
　└ 週150分以上・持続可能な様式選択（歩行・水中・自転車）
　
マインドボディ（短期機能改善）＋ニューロモーター（歩行改善）の補完
```

### 半月板病変の保守療法優先ロジック
```
変性型 → PTが第一選択（論文③④：ネットワークメタ分析レベル証拠）
外傷性（若年）→ 12週の監視下運動で手術と同等（論文⑤：DREAM Trial）
ロッキング・保存療法失敗→ 外科的判断へ
```

### PFPの個別化処方
```
股関節筋力低下 → Hip-and-Knee複合（論文②）
疼痛破局化↑・BMI≥25 → 股関節優先（論文②二次解析）
ベースライン疼痛強度高 → 膝強化優先
```

### TKA後リハゴール設定（論文⑦より）
```
術後0-6週: ROM・除痛・基本動作獲得
術後6-12週: 筋力強化・ADL拡大  
術後12-24週: 実用的機能回復・スポーツ復帰準備
```
