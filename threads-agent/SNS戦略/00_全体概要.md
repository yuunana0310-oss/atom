# Threads × AI 完全自動運用システム 全体概要

## 実績
- 2週間 × 完全自動（0%人間）: **30,000円**
- 4日 × 5%人間モード: **420,000円（13倍）**

## なぜThreadsか

| SNS | フォロワー0からバズれるか | テキストだけでOKか | AI自動化しやすいか |
|---|---|---|---|
| X（Twitter） | ✗ フォロワー必須 | ✓ | △ |
| Instagram | △ | ✗ 画像必須 | ✗ |
| TikTok | △ | ✗ 動画必須 | ✗ |
| **Threads** | **✓ 新規でも拡散** | **✓** | **✓ 最強** |

→ テキストのみ・アルゴリズム拡散・AI相性最強で自動化一択

## 2つの運用モード

| | 完全自動（0%人間） | 5%人間モード |
|---|---|---|
| 投稿ルート | Writer → キュー → 投稿 | Writer → 下書き → 人間10分レビュー → キュー → 投稿 |
| 作業時間 | ほぼゼロ | 1日10分 |
| 投稿品質 | 7点を10本 | 9点を5本 |
| 向いてる人 | 複数アカウント並行・放置したい | 1アカウントを本気で伸ばす |
| 収益感 | 月3〜5万/アカウント | 桁が変わる |

## マネタイズ構造

```
投稿（有益コンテンツ・売り込みなし）
  └── コメント欄にASPアフィリエイトリンクを自動配置
        └── 承認率保証案件を優先 → 収益が安定
```
