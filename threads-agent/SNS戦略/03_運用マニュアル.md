# 毎日の運用マニュアル

## 完全自動モード（0%人間）

### 初回起動
```bash
cd C:\Users\yuuna\agens\threads-agent
python main.py --agent all --mode auto
```

### 毎日の運用
`run.sh` をcronに登録するだけ。あとは何もしない。

**Windowsでの自動実行（タスクスケジューラ）:**
1. タスクスケジューラを開く
2. 「タスクの作成」
3. トリガー: 毎日 朝8時
4. 操作: `python C:\Users\yuuna\agens\threads-agent\main.py --agent all --mode auto`

---

## 5%人間モード（推奨）

### 毎日のルーティン（合計10〜15分）

**朝（自動・放置でOK）**
```bash
python main.py --agent all --mode review
```
↑ これをcronで自動実行。リサーチ・分析・投稿生成が走り、下書きが溜まる。

**隙間時間10分（ここだけ人間）**
```bash
python main.py --agent review
```

レビュー画面の操作:
```
[a] 承認 → キューに追加（投稿される）
[s] スキップ → 見送り
[e] 1行目を編集 → 修正してから再判断
[q] 中断
```

判断の基準:
- 「1行目でスクロールが止まるか」
- 「今のタイミングで出して刺さるか」（トレンドの空気感）
- 「似たような投稿を最近出してないか」

**自動投稿（放置）**
レビューでOKした投稿が時間スロットに沿って自動で出ていく。

## コマンド一覧

```bash
# 全体実行
python main.py --agent all --mode review    # 5%人間モード
python main.py --agent all --mode auto      # 完全自動

# 個別実行
python main.py --agent researcher   # ネタ収集のみ
python main.py --agent analyst      # 分析のみ
python main.py --agent writer --mode review  # 下書き生成のみ
python main.py --agent review       # 下書きレビューUI
python main.py --agent poster       # 投稿実行のみ
python main.py --agent fetcher      # メトリクス取得のみ

# 監視・管理
python main.py --agent supervisor-status    # 現在の状態確認
python main.py --agent kill-switch-on       # 緊急全停止
python main.py --agent kill-switch-off      # 停止解除
python main.py --status                     # APIキー確認
```

## 投稿スロット

| 時刻 | 備考 |
|---|---|
| 08:00 | 朝の通勤時間 |
| 10:00 | 午前中 |
| 12:00 | 昼休み（高エンゲージメント） |
| 14:00 | 午後 |
| 16:00 | 夕方前 |
| 18:00 | 退勤時間（高エンゲージメント） |
| 20:00 | 夜（高エンゲージメント） |
| 21:30 | ゴールデンタイム |
| 23:00 | 夜更かし層 |
| 01:00 | 深夜 |

- 1日最大10投稿
- 最低間隔1時間（bot判定回避のため絶対に変えない）

## 収益確認タイミング

ASPアフィリエイトの場合、成果発生から確定まで30〜60日かかるのが一般的。
投稿4〜7日後にクリック数・承認数をASP管理画面で確認する。
