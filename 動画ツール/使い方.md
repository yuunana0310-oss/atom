# スライドショー動画ツール（4:5 縦・フィード投稿向け）

複数画像 ＋ キャプション → 字幕付きスライドショー動画(mp4) を一発生成。

## 使い方（3ステップ）

1. **画像を入れる**
   `input/` に画像を入れる。並ぶ順にしたいので `01_`, `02_` … と先頭に番号を付けると確実。

2. **キャプションを書く**
   `input/slides.txt` に「画像ファイル名 | キャプション」を1行ずつ書く。
   ```
   01_xxx.png | 歩ける数値、歩けない心
   02_xxx.png | だから私は記録のしかたを変えた
   03_xxx.png | 今日からできる、たった一つの工夫
   ```
   - キャプション内で改行したいときは `\n` を入れる。
   - `slides.txt` が無い場合は `input/` の画像を名前順・字幕なしで使う。

3. **実行**
   ```
   cd C:\Users\yuuna\agens\動画ツール
   python make_video.py
   ```
   → `output/result.mp4` が完成。

## オプション

| オプション | 意味 | 例 |
|---|---|---|
| `--sec` | 1枚あたりの表示秒数（デフォルト3） | `--sec 4` |
| `--fade` | 切り替えのクロスフェード秒数（デフォルト0.5） | `--fade 0.8` |
| `--out` | 出力ファイル名 | `--out threads01.mp4` |

例: `python make_video.py --sec 4 --out threads01.mp4`

## 仕様メモ

- 出力サイズ: **1080×1350（4:5）/ 30fps / H.264 mp4**
- 画像は中央にフィット表示、余白は元画像のぼかしで自動的に埋める
- フォント: メイリオ太字（縁取り付きで無音再生でも読める）
- ffmpeg は winget 版を自動検出（PATH 未設定でも動く）

## あとから足せること（必要になったら）

- VOICEVOX/AivisSpeech でナレーション音声を追加（字幕と同期）
- BGM のミックス
- 9:16 / 16:9 / 1:1 など他サイズ出力（W,H を変えるだけ）
