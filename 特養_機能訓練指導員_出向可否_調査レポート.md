## 調査概要

**質問の背景**：
- 当院のあん摩マッサージ指圧師（月～土、週30時間勤務）が、同法人の特養で機能訓練指導員として出向できるか
- 午前中のみなど、パートタイム配置は法的に問題ないか
- 2021年の法改正により配置要件がどう変わったか

**調査対象**：
- 令和3年度（2021年度）介護保険法改正
- 厚生労働省 Q&A資料
- 特別養護老人ホームの人員配置基準

## 🔄 2021年介護保険法改正のポイント

### ✅ 最も重要な変更点：**機能訓練指導員の常勤要件が撤廃された**

#### 改正前（2020年まで）
- 機能訓練指導員は**常勤職員**として配置することが基本的な要件

#### 改正後（2021年4月以降）
- **常勤要件が廃止** → 非常勤・パートタイム配置が可能に
- ただし、**加算算定時の条件として常勤要件が復活**する場合あり

## 👤 機能訓練指導員の資格要件

### あん摩マッサージ指圧師は該当する資格である ✅

特別養護老人ホームの機能訓練指導員として認められる資格：

| 資格種別 | 要件 |
|---------|-----|
| **理学療法士** | 資格取得者 |
| **作業療法士** | 資格取得者 |
| **言語聴覚士** | 資格取得者 |
| **看護職員** | 看護師、准看護師 |
| **柔道整復師** | 資格取得者 |
| **🎯 あん摩マッサージ指圧師** | **資格取得者** |
| **鍼灸師** | 資格取得者 + 6ヵ月以上の実務経験 |

### あん摩マッサージ指圧師としての役割
- 利用者の体の違和感（肩こり、首のこり、腰痛、筋肉のハリなど）をマッサージや指圧療法で軽快・軽減
- 手技によるリハビリテーション

## 📊 特養での人員配置基準

### 基本的な配置基準

| 要素 | 基準 |
|------|------|
| **必須人数** | 1名以上 |
| **常勤要件** | **基本的にはなし**（2021年改正） |
| **他職務との兼務** | **可能** |
| **非常勤配置** | **可能**（基本的には） |

### ⚠️ 「個別機能訓練加算」算定時の追加条件

特養が**個別機能訓練加算**を算定する場合には、以下の厳しい条件が課される：

#### 個別機能訓練加算（Ⅰ）ロの場合
- **入所者が100名を超える場合**：
  - 常勤専従の機能訓練指導員を**1名以上**配置
  - 加えて、常勤換算法で「入所者数÷100」以上を配置

#### これは何を意味するか？
- 加算を算定しない場合 → **パートタイム・非常勤配置 OK**
- 加算を算定する場合 → **常勤・専従配置が必須**

## ✅ パートタイム・非常勤配置の可否

### 【結論】**基本的には可能**だが、条件付き

#### ケース1：特養が個別機能訓練加算を「算定していない」場合
✅ **パートタイム出向が可能**
- 法的制約なし
- 午前中のみ、週2～3日など、柔軟な勤務形態が可能

#### ケース2：特養が個別機能訓練加算を「算定している」場合
❌ **パートタイム出向は不可（加算要件違反）**
- 常勤・専従配置が必須
- パートタイムに変更すると加算の算定要件を満たさなくなる

## 🎯 結論

### あなたのケースで出向が可能かどうか

| 条件 | 結果 | 根拠 |
|------|------|------|
| **資格要件** | ✅ **満たしている** | あん摩マッサージ指圧師は機能訓練指導員の資格要件に該当 |
| **配置形態** | **条件付き OK** | 2021年改正で常勤要件が廃止 |
| **パートタイム可否** | **加算状況に左右される** | 加算なし→OK、加算あり→NG |

### **最優先で確認すべき事項**

**🔴 次のステップ**：対象特養に確認してください

1. **「現在、個別機能訓練加算を算定していますか？」** ← これが全てを決定します

   - **「算定していない」の場合**  
     → あなたのパートタイム出向は法的に問題なし
   
   - **「算定している」の場合**  
     → パートタイム出向は不可。常勤専従配置が必須。
     
2. 「加算を算定する予定はありますか？」  
   → 近い将来の算定予定も確認が必要

## ⚠️ 注意点・確認すべき事項

### 1. **同一法人内での出向形態**
- 本人が両職場に登録される「兼務」形式
- または、給与や時間管理は当院で行い、業務時間のみ出向する形式
- **いずれでも法的には問題ありません**（配置基準上は同等）

### 2. **労働契約上の確認**
- 当院との雇用契約内容（給与、社保など）
- 特養への出向にあたり、新たな契約は必要か
- 同法人内なので、手続きは簡素化される可能性が高い

### 3. **常勤換算職員数への計上**
- 法人全体で「常勤換算職員数」を管理している場合  
- 週30時間勤務は「常勤換算」では何%として計上されるか確認
- （一般的には常勤を週40時間と見なした場合、30時間なら75%）

### 4. **実務経験について**
- あん摩マッサージ指圧師で「鍼灸師ではない」ため  
- 6ヵ月実務経験要件はあなたには該当しません ✅

### 5. **その他の兼務との兼ね合い**
- 当院での現在の業務との兼ね合い
- 週30時間 + 特養での新規業務のバランス
- 過労や業務品質低下のリスク管理

## 📌 推奨される確認フロー

```
Step 1: 対象特養に「個別機能訓練加算の算定状況」を確認
   ↓
Step 2a: 算定していない → パートタイム出向が可能（下記へ）
   ↓
Step 3: 以下を法人・特養で協議
   - 出向形態（兼務か配置転換か）
   - 勤務日時・時間数
   - 給与・待遇
   - 法人内での人事手続き
   ↓
Step 4: 実装
```

**Step 2b: 加算を算定している場合**
- パートタイム出向は不可
- 代替案を検討（常勤配置など）

## 📞 さらに質問・確認が必要な場合

以下の項目について、対象特養の管理者・事務長に確認されることをお勧めします：

- [ ] 現在の加算算定状況（個別機能訓練加算Ⅰ・Ⅱ・Ⅲの別）

- [ ] 想定される勤務形態（週何時間、いつ出向するか）
- [ ] 給与・待遇の扱い
- [ ] 同法人内での人事手続きの流れ
