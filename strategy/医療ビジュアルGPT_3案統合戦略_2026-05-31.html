医療ビジュアルGPTパック 3案統合戦略レポート

作成日 2026-05-31 ／ 対象：あとむ（現役PT × Clinical Visual Communication）／ 用途：着手前の全体像確定
結論（先に動く順）： ②患者説明イラストGPT（本命）を主商品として完成・販売 → ①SNS・サービス訴求ビジュアルGPTを発信導線兼ライト版として接続 → ③院内勉強会GPTを高単価・後発として展開。
3つを別商品として散らさず、「医療情報を伝わる形へ設計し直す」1つの世界観で束ねる。

1. 商品3案の全体像

3案はいずれも「設計済みのAI活用フレーム」。共通エンジン（ヒアリング→用途・対象判定→情報密度調整→失敗しにくい出力）を持ち、出力先と対象が違うだけ。だから1本ずつ作るのではなく、本命を軸に派生させる。

本命 / 最優先

② 患者説明イラストGPT

患者・家族向けの説明図／自主トレ図／退院指導図を、その患者専用に量産。安心感重視・不安を煽らない設計。

対象：患者・家族に説明する医療職（PT中心）
差別化軸：forPT等の素材サイト＝「探して選ぶ」に対し、本商品＝「その患者向けに作る」

導線 / 早期接続

① SNS・サービス訴求ビジュアルGPT

Threads/IG/note/LP用の、目に止まる構図のビジュアル設計。AI臭排除・主役1点設計。

対象：発信・集客したい医療職／自院PR
役割：あとむ自身の発信エンジンでもあり、商品でもある（二役）

高単価 / 後発

③ 院内勉強会・医療者向け学習資料GPT

評価/介入/リスクを整理した高密度インフォグラフィック・スライド構成・台本生成。

対象：院内教育担当・指導的立場の医療職
役割：実績が貯まってから出す上位商品（単価を取りに行く層）

2. 価格設計

商品初期価格値上げ方針正規到達価格根拠
② 患者説明イラストGPT（本命） 2,980円 5部ごとに+500円 4,980円 実用GPT＋プロンプト全文。相場（2,980〜）の中核帯
① SNS・サービス訴求GPT 無料〜1,980円 無料版で認知→有料版へ 2,980円 導線役。無料ライト版で信頼構築、本命へ送客
③ 院内勉強会GPT 4,980円 実績後に+1,000円帯 7,980円〜 高密度・高単価。法人/指導層向けで価格耐性あり
★ 3点セット（バンドル） 後日設定 本命の実績が出てから 9,800円前後 3案が揃った段階で「設計フレーム全部入り」として提示

原則：体験談・日記は無料、再現性ある実用情報（プロンプト全文・ヒアリング設計・失敗の立て直し）が有料コアの価値。煽り・誇大価格にしない。

3. 導線（コンバージョン）設計

[Threads / Instagram 発信（発信軸6本）] │ ① のビジュアルで「作例・Before/After」を見せる ▼ [note プロフィール固定 / 無料記事・無料ライト版GPT(①)] │ 「探すより作るほうが速い」を体験させる ▼ [本命② 販売LP（無料プレビューで購入判断）] │ forPT差別化 → 設計プロンプト予告 → 価格・早期特典 ▼ [購入 → 導入ガイド + 30日実装ロードマップ] │ 使えた実感 → 感想シェアを促す ▼ [購入者の発信 → 新規認知] ＋ [③・バンドルへアップセル]

最重要：LPの無料プレビューでほぼ購入が決まる。有料部分を読ませる前に「これは自分のためのものだ」と感じさせる。①は売上より送客と信頼構築を主目的にする。

4. 着手順（フェーズ分解）

フェーズ1：本命②の土台を完成させる（今ここ）

#タスク状態
1販売LP記事の作成（forPT差別化を前面に）✅ 作成済
2GPT本体の設計プロンプト・用途別プロンプトの整備とテスト次の一手
3作例・失敗例・改善例の追加（信頼の核）未
4購入者向け導入ガイド ＋ 30日実装ロードマップ未
5note下書き投稿 → 目視確認 → 公開（有料記事ルール遵守）未

フェーズ2：発信軸を安定させ、①を導線として接続

  • Threads発信軸6本を回す（特に「Before/Afterの視覚的証明」「現場課題」）
  • ①の無料ライト版 or 作例を発信に組み込み、本命②へ送客
  • 購入者の感想（社会的証明）を発信に反映

フェーズ3：③高単価とバンドルで単価を上げる

  • ②の実績・型が固まってから③を制作（高密度インフォグラフィック設計）
  • 3案揃った段階でバンドル化、上位顧客への提案
  • その後に海外版（Clinical Visual System／Substack）再構成へ
優先順位の歯止め：新しい案（③・バンドル・海外版）に着手する前に、必ず「本命②の薄い箇所が埋まっているか」を確認する。方向性を増やしすぎる傾向・完璧主義による公開遅延が最大のボトルネック。本命を売れる状態にすることが、全導線の土台。

5. 3案を貫く1つのメッセージ

「医療情報を、ただ正しく並べるのではなく、患者・家族・医療者に届く形へ設計し直す。」
②=患者・家族へ、①=世の中へ、③=医療者へ。届ける相手が違うだけで、設計思想は1つ。これが他のAI教材・素材サイトとの本質的な差別化点。
