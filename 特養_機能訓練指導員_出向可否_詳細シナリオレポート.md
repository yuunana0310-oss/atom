## はじめに

あなたのケース：
- **あん摩マッサージ指圧師**（国家資格者）
- 現在：月～土、週30時間勤務（当院）
- 希望：同法人の特養でパートタイム出向して機能訓練指導員として勤務

**質問の本質**：
> パートタイムでの出向が「法的に」可能か？

**答え**：
> 対象特養が「個別機能訓練加算」を算定しているかどうかで、180度異なります

---

## 背景：2021年法改正とは何か

### 改正の歴史

```
2020年まで：機能訓練指導員は「常勤が原則」
   ↓
2021年4月（令和3年度）：改正
   ↓
現在：常勤要件は廃止されたが「加算を取る場合のみ常勤が復活」
```

### 何が変わったのか？

**改正前（2020年まで）**
- 機能訓練指導員は常勤職員でなければならない
- パートタイムや非常勤配置は認められない

**改正後（2021年4月以降）**
- 基本的な配置基準では常勤要件が廃止
- パートタイム・非常勤配置が可能に
- ただし「加算を算定する場合」は常勤要件が復活

# シナリオA：加算を「算定していない」場合

## ✅ パートタイム出向が完全に可能

### 法的な立場

特養が**個別機能訓練加算を算定していない**場合、機能訓練指導員の配置基準は：

| 項目 | 要件 |
|------|------|
| **配置人数** | 1名以上 |
| **勤務形態** | 常勤・非常勤・パートタイム **すべて可能** ✅ |
| **配置時間** | 毎日配置の必要なし |
| **他職務との兼務** | 可能 |

### あなたのパートタイム出向で具体的に何ができるか

#### パターン1：午前中のみ出向（推奨）
```
月～金：午前（例：8:00～12:00）
土曜：午前（例：8:00～12:00）

時間数：週20時間程度
→ 当院での週30時間 + 特養での週20時間 = 週50時間（兼務）
```

**メリット**：
- 当院と特養の両方で機能訓練業務を担当可能
- スケジュール調整が容易
- 実務経験を1.5倍積める

**問題点**：
- 両職場での移動時間が必要
- 過労管理が重要

---

#### パターン2：週2日の専従出向
```
月・水：終日（例：8:00～17:00）
その他：当院で週30時間

時間数：週16時間（特養）+ 週30時間（当院）= 週46時間
```

**メリット**：
- 特養での業務に集中できる
- 移動時間が少ない

---

#### パターン3：非常勤・スポット出向
```
特定の曜日・時間のみ出向
例：水曜午後（週4時間）など

時間数：週4～10時間程度
```

**メリット**：
- 柔軟性が最も高い
- 試験的運用に最適

---

### あん摩マッサージ指圧師としての活動内容

加算を算定していない特養での業務：

- **利用者への手技療法**：マッサージ、指圧
- **肩こり、腰痛などの緩和**：日常的なケア
- **簡易的な運動指導**：拘縮予防、可動域保持
- **利用者の個別対応**：必要に応じた施術

⚠️ **注意**：
- 「個別機能訓練計画書」の作成は不要
- 個別のスコア評価（SIAS、Barthel Indexなど）の記録は不要

# シナリオB：加算を「算定している」場合

## ⚠️ パートタイム出向は不可だが、代替案はある

### 法的な立場

特養が**個別機能訓練加算を算定している**場合、機能訓練指導員の配置基準は：

| 項目 | 要件 |
|------|------|
| **配置人数** | 入所者に応じた人数 |
| **勤務形態** | 常勤・**専従**が必須 ⚠️ |
| **配置時間** | 毎日、営業時間全体 |
| **他職務との兼務** | **不可**（加算算定時） |

### なぜ常勤が必須なのか？

加算の公式が：

```
「専ら機能訓練指導員の職務に従事する
 理学療法士または作業療法士などを配置する」
```

と定められているためです。

「専ら」= **その職務だけに専念する**という意味。  
つまり、他の職場との兼務は不可。

---

### 加算の種類と常勤要件

#### 個別機能訓練加算（Ⅰ）ロ

**対象施設**：入所者が100名を超える特養

**常勤要件**：
- 常勤専従の機能訓練指導員を1名以上
- 加えて、常勤換算で「入所者数÷100」以上を配置

**例**：入所者150名の場合
```
常勤専従：1名
常勤換算：150÷100 = 1.5名以上
→ 合計：常勤2.5名分以上必要
```

**加算額**（参考）：
- 個別機能訓練加算（Ⅰ）ロ：要介護度別に日額60～100単位
- 月換算：要介護度別に月額1,800～3,000円

---

#### 個別機能訓練加算（Ⅱ）

**対象施設**：入所者が100名以下の特養

**常勤要件**：
- 常勤換算で入所者30名につき1名以上

**例**：入所者60名の場合
```
常勤換算：60÷30 = 2名以上必要
```

**加算額**（参考）：
- 日額30～50単位
- 月換算：月額900～1,500円

---

### あん摩マッサージ指圧師はこの加算の対象か？

**はい、対象です。**

加算の要件では「理学療法士、作業療法士、言語聴覚士、**看護職員、柔道整復師、あん摩マッサージ指圧師**」と明記されています。

つまり、あなたが常勤で配置されれば、加算の要件を満たします。

---

### シナリオB：出向は不可だが、代替案はある

#### 代替案1：「常勤配置」への配置転換

```
現在：当院で週30時間（非常勤・パートタイム）
  ↓
変更後：特養に常勤配置（週40時間）

メリット：
✅ 加算の要件を満たす
✅ 特養での責任ある業務が可能
✅ 給与・福利厚生が充実する傾向

デメリット：
❌ 当院から「転出」扱い（完全配置転換）
❌ 当院でのマッサージ業務が終了
```

---

#### 代替案2：「常勤換算」による配置

```
当院での勤務：週30時間（非常勤）
特養での勤務：週30時間（非常勤）

合計60時間を「常勤換算」で処理

例：週40時間を常勤として見なす場合
当院：30時間 = 0.75常勤換算
特養：30時間 = 0.75常勤換算
      → 特養側では「常勤換算1.5名」に計上

可能か？
→ 法人内で常勤換算を柔軟に解釈している場合は可能
→ ただし「常勤専従」という文言との齟齬がある
→ 厚生労働省の指導は「常勤かつ専従」が原則
```

⚠️ **注意**：
この方法は「グレーゾーン」です。監査で指摘される可能性があります。

---

#### 代替案3：加算算定の「廃止」を検討する

```
現在：個別機能訓練加算を算定している
  ↓
変更後：加算を算定しない方針に変更

メリット：
✅ パートタイム出向が可能になる
✅ あなたの配置が実現できる
✅ 人員配置の柔軟性が増す

デメリット：
❌ 特養の経営が苦しくなる（加算がなくなる分）
❌ 月額で900～3,000円の減収（入所者数により異なる）
❌ 既存の利用者への個別計画書作成義務が廃止

実現性：
→ 加算の効果が大きい場合、廃止は困難
→ 「加算を廃止してでもあなたを採用したい」という
   特養の強い意思が必要
```

---

### シナリオB：加算算定時の法的問題

#### 厚労省の解釈

```
Q: パートタイムの機能訓練指導員で
   加算を算定できますか？

A: いいえ。
   「常勤かつ専従」という要件から、
   パートタイムでの算定は不可です。

   もしパートタイムで加算を算定した場合、
   不正請求となり、返金請求の対象になります。
```

**出典**：  
[令和３年度介護報酬改定Q&A - 個別機能訓練加算について](https://tsuusho.com/news/article/%E4%BB%A4%E5%92%8C%EF%BC%93%E5%B9%B4%E5%BA%A6%E4%BB%8B%E8%AD%B7%E5%A0%B1%E9%85%AC%E6%94%B5%E5%AE%9A%E3%81%AB%E9%96%A2%E3%81%99%E3%82%8BQ%EF%BC%86A)

### シナリオB：実装手順（加算廃止ルート）

#### Step 1：特養管理者との面談

```
「個別機能訓練加算を算定されているようですが、
パートタイム出向での配置を希望しています。
以下の点についてご相談させていただきたいのですが…」

☑ 加算廃止の可能性は検討可能か

☑ 利用者への説明はどうするか
☑ 廃止の手続き（厚生労働省への届け出など）
```

#### Step 2：加算廃止の影響試算

```
月額減収の試算：
（加算額 × 対象利用者数 × 月営業日数）
```

**例**：
- 加算額：1日50単位 = 500円
- 対象利用者：60名
- 月営業日：26日

```
500円 × 60名 × 26日 = 780,000円/月の減収
年間：約930万円の減収
```

⚠️ **この金額が許容できるかは法人経営層の判断**

---

```
加算廃止を決定した場合：

☑ 利用者・家族への説明

☑ ケアプラン・個別計画書の変更
☑ 介護職員への説明
☑ あなたの配置と業務内容の決定
```

## 加算の有無を見分けるポイント

### 確認方法1：特養の管理者に直接聞く（最確実）

```
「現在、個別機能訓練加算を算定されていますか？」

Yes → シナリオB（加算あり）
No → シナリオA（加算なし）
```

---

### 確認方法2：間接的な確認（自分で推測する）

#### サイン1：「個別機能訓練計画書」が存在するか

- あれば → **加算の可能性が高い**
- ケアプランと別に、入所者ごとに「個別機能訓練計画書」を作成しているか確認

#### サイン2：専従の理学療法士・作業療法士がいるか

- いれば → **加算の可能性が高い**
- 「理学療法士を常勤で配置している」と聞いたことがあるか

#### サイン3：入所者が100名前後以上か

- 多い特養 → 加算（Ⅰ）ロの可能性
- 少ない特養 → 加算（Ⅱ）または加算なしの可能性

#### サイン4：法人内で「機能訓練専任者」という職制があるか

- あれば → 加算をかなり意識している
- なければ → 加算なしの可能性が高い

---

### 確認方法3：特養の財務情報から推測（上級）

```
特養の運営事業報告書やホームページで
「個別機能訓練加算」の記載がないか確認

大法人なら公開情報の可能性あり
```

### シナリオB（加算あり）の場合

最初の協議で伝えること：
```
「パートタイム出向を希望していますが、
加算算定時には常勤要件があると伺いました。
以下の点について、ご相談させていただきたいのですが…

1. 常勤配置への転出が可能か
2. 加算廃止の可能性はあるか
3. 常勤換算兼務の可能性はあるか」
```
