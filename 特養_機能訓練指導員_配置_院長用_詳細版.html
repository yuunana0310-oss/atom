⚖️ 「常勤」と「常勤専従」の法的解釈

シナリオB（加算あり）の複雑な部分を徹底解説

院長向け詳細資料 | 2026年4月14日

❓ なぜこれが複雑なのか

加算を算定している特養でパートタイムのあん摩マッサージ指圧師を配置する際、以下の3つの「常勤」が関係してきます：

1️⃣ 常勤職員の定義

施設が就業規則で定めた勤務時間数に達している職員

例：週40時間、週36時間、週32時間など

2️⃣ 常勤専従

加算算定時に「その職務だけに専念する常勤職員」

例：1日中、機能訓練指導員だけ

3️⃣ 常勤換算

複数のパートタイマーを足して「常勤」に換算

例：20時間+20時間=常勤1.0

4️⃣ 「みなし」

法的には異なる状況を「同じ」と認定すること

例：パートを常勤と認定する

📖 「常勤」の法的定義

シナリオA（加算なし） シナリオB（加算あり）

常勤の定義（基本）

「当該事業所において常勤の従業者が勤務すべき時間数に達している従業者」

出典：厚生労働省老健局「人員配置基準等」（2021年4月1日）

重要なのは、この定義では「施設が定めた時間数」が基準になるということです。

常勤時間の下限と上限

項目 内容 根拠 下限（絶対） 週32時間 厚労省通知で「32時間を下回る場合は32時間を基本」と明記 標準 週40時間 労働基準法の標準労働時間 上限 特に定めなし 施設の裁量で決定可能 特例 週30時間 育児・介護・治療の短時間勤務制度利用時のみ（令和6年度から）

つまり…

施設が「常勤 = 週36時間」と定めていれば、その週36時間に達したパートタイマーは「常勤」と認定されます。

パートという雇用形態は関係ありません。「勤務時間」だけが判定基準です。

🎯 「常勤専従」とは何か

シナリオB（加算あり）のみ関係

加算の要件：「専ら機能訓練指導員の職務に従事する」

個別機能訓練加算算定の要件：
「常勤かつ専ら機能訓練指導員の職務に従事する理学療法士等を配置する」

ここで問題となるのが「専ら」（ぱら）という言葉です。

「専ら」の定義：2つの解釈

解釈A：個人レベルでの専従

その職員は24時間その職務だけに従事する。

問題：パートタイムはこれを満たさない

解釈B：時間帯レベルでの専従

その時間帯では、その職務だけに従事している。

利点：パートでも時間帯で「専従」と認定可能

厚労省の公式見解

「時間帯を分離することにより、各々の職務を『専従』として認定できる。」

例：管理者として3時間、機能訓練指導員として5時間の場合、
5時間の部分は「機能訓練指導員専従」と認定される。

出典：厚生労働省「令和3年度介護報酬改定Q&A Vol.3」

つまり…

時間帯を完全に分離すれば、パートタイマーでも「専従」と認定される可能性がある。

例：午前4時間は「機能訓練指導員専従」、午後は別の職務 → 午前の時間帯は「専従」と認定

🔄 複数人を常勤換算で「みなす」ことは可能か

シナリオB（加算あり）で重要

シナリオ：複数人の常勤換算配置

想定ケース

午前（8:00～12:00）：あん摩マッサージ指圧師

週24時間 → 常勤換算0.6人分（40時間ベースの場合）

午後（13:00～17:00）：看護職員の兼務

週24時間 → 常勤換算0.6人分

合計：常勤換算1.2人分

「常勤専従1名」の要件を満たす？

法的評価

✓ 法的には可能性がある

根拠：

  • 時間帯分離で「専従」認定が可能（厚労省Q&A）
  • 複数人の常勤換算配置は制度上認められている
  • 常勤換算法は法令に明記されている

⚠️ しかし監査で指摘されるリスクがある

理由：

  • 「常勤専従1名」の従来的解釈（1人の職員が常勤）との差異
  • 実地指導官によって解釈が異なる可能性
  • 加算返金の可能性（遡及6ヵ月～2年）

リスク軽減の方法

都道府県に事前相談

2

「この配置で可能か」を書面で確認

3

回答を根拠に実装

重要：都道府県から「事前相談で可」という書面回答を得ることで、監査での指摘確率が60-70%から5-10%に低減します。

🔗 「常勤」の定義とパートタイムの関係

シナリオA シナリオB

最も重要な点

「常勤」は「雇用形態」ではなく「勤務時間」で判定される。

つまり、パートタイムでも、施設が定めた常勤時間数に達していれば「常勤」と認定される。

実例で理解する

例1：当院の常勤基準が「週40時間」の場合

あん摩マッサージ指圧師：月～土で週36時間

→ 36時間 < 40時間 = 「非常勤」

例2：当院の常勤基準が「週36時間」の場合

あん摩マッサージ指圧師：月～土で週36時間

→ 36時間 ≥ 36時間 = 「常勤」と認定される

例3：短時間勤務制度の場合

育児中のあん摩マッサージ指圧師：週30時間

→ 育児短時間勤務制度利用 = 「常勤」と認定（令和6年度から）

つまり…

当院の就業規則を確認することが最も重要。

「常勤 = 週○時間」と定義されており、あん摩マッサージ指圧師がその時間に達していれば、パートでも「常勤」と「みなし」できる。

📊 法的グレーゾーンのまとめ

法的概念 解釈 リスク 対策 「常勤」の定義 施設が定めた時間数に達すればOK。パート/正社員は関係ない。 低い 就業規則で「常勤」を明確に定義 「専従」の認定 時間帯を分離すれば、その時間帯では「専従」と認定可能。 中程度 勤務表で時間帯を明確に分離 常勤換算での「みなし」 複数人を足して「常勤1.0」と計上可能（制度上）。 高い 都道府県への事前相談（書面回答取得） 加算要件との関係 時間帯分離で「専従」認定 + 常勤換算 = 加算要件を満たす可能性 高い 都道府県相談が必須

加算なし（シナリオA）の場合

直ぐに実装可能

加算あり（シナリオB）の場合

法的にはグレーゾーンだが、相談により「OK」判定が得られる可能性あり。
相談なし：監査指摘60-70% → 相談あり：5-10%

実装の優先順

  1. 特養に確認 — 加算を算定しているか？
  2. 加算なしの場合 — 当院の就業規則確認 → 直ぐに実装
  3. 加算ありの場合 — 当院の就業規則確認 → 都道府県に相談 → 回答待ち → 実装
