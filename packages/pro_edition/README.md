## 🌟 パッケージ概要
本パッケージは、単なるAIツールではなく、士業事務所や中小企業の「知能インフラ」を構築する自律型プラットフォームです。
最新のe-Gov API V2から取得した膨大な法令データに加え、**民法・商法・会社法などの基本六法**、そして実務に即した**判例・裁決事例**をローカル環境で学習済みです。

## 📚 圧倒的な知識量（Ultimate RAG）
- **初期知識**: 5,700件超（法令全文 + 実務ハンドブック）
- **追加学習済みパック**:
  - **リーガル・パック**: 日本国憲法、民法、商法、会社法、刑法、刑事訴訟法、民事訴訟法
  - **判例・事例パック**: 労働・税務・民事に関する主要判例・裁決事例

## 🛠️ 製品ティア（販売モデル）
本システムは、ターゲットに合わせて以下の3つの構成で展開可能です。
1. **Basic版**: 4名体制、税務・労務の基本データのみ。
2. **Pro版 (標準)**: 6名体制、基本六法（リーガル・パック）搭載。
3. **Ultimate版**: 6名体制、基本六法 + 判例データベース搭載。

## 🚀 始め方
1. `Start_Elite_Team.bat` をダブルクリックしてください。
2. ブラウザで **「Pro Intelligence Platform」** が起動します。
3. セキュアな環境で、6人のエリートチームと業務を開始してください。
