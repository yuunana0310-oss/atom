# 副業でAIを使い始めて半年。収入が増えた部分と、想像と違った部分を正直に書く

「AIを使えば副業収入が爆発的に増える」という話、どこかで見たことはないだろうか。
自分もそう信じて半年前に始めた。結果は——期待通りの部分もあれば、全然違う部分もあった。

正直に言うと、最初の2ヶ月はうまくいかなくて、**少し焦った**。それでも続けてよかったと今は思っている。

---

## はじめに：この記事を書いた人について

フリーランスのWebライター兼ディレクターとして活動して4年。月収は長らく8〜12万円の間で停滞していた。

「これ以上は時間的に無理だ」と壁にぶつかっていたタイミングで、AIを副業に本格導入した。半年間で試したツールは7種類以上、受注した案件は30件超。収入の変化は月平均+4〜5万円ほど。

ただ、それは「AIが全部やってくれた」結果ではない。そこに至るまでの試行錯誤と、正直な失敗も含めて書いていく。

---

## この記事でわかること

AIを副業に使い始めようとしている人、あるいはすでに使っているけど「なんか思ったより稼げない」と感じている人に向けて書いた。

## 1. AIを副業に使い始めたきっかけ

月収12万円の壁は、単純に「時間」の問題だった。

1件の記事に3〜4時間かけていた。単価を上げようにも、クライアントとの交渉は難しい。時間を増やそうにも、本業との兼ね合いで限界がある。

そこに出てきたのが「ChatGPTで記事を10倍速く書ける」という話だった。半信半疑だったけど、試さない理由はなかった。

だから使い始めた。半信半疑のまま。

---

## 2. 収入が増えた部分：正直な数字と内訳

半年前と比較した収入の変化はこうだ。

| 副業カテゴリ | 半年前 | 現在 |
|---|---|---|
| Webライティング | 月7〜8万円 | 月10〜11万円 |
| 構成・ディレクション | 月2〜3万円 | 月4〜5万円 |
| noteコンテンツ販売 | 0円 | 月1〜2万円 |
| **合計** | **月9〜11万円** | **月15〜18万円** |

増えた要因は「量をこなせるようになった」こと、そして「ライティング以外の仕事も受けられるようになった」ことの2つ。

ライティングの速度は体感で2〜3倍になった。ただし「AIが書く」ではなく「AIと一緒に書く」に近い。この違いが最初の誤算でもあった。

---

## 3. 想像と違った部分：3つの誤算

**誤算①：AIが書いてくれるわけではない**

「プロンプトを入れれば記事ができる」と思っていた。実際は、AIが出してきたものをそのまま使えることはほぼない。事実確認、文体の修正、読み手への配慮——その作業がなくなるわけではない。

変わるのは「草稿を作る時間」がゼロに近づくこと。それだけでも十分価値はあるが、「楽になる」より「速くなる」という感覚に近い。

これに気づいたとき、**少し拍子抜けした**。でも同時に「じゃあ速くなった分で何をするか」を考えるようになって、仕事の幅が広がった。

**誤算②：クライアントによっては使えない**

一部のクライアントは「AI生成禁止」を明示している。また明示していなくても、AI特有の文体を嫌がるケースもある。

最初はすべての案件でAIを使おうとして、2件ほど問題になった。今は使用可否をヒアリングするのが当たり前になった。

**誤算③：ツール費用がかかる**

無料枠だけでは仕事にならない。月に使うAPIやサービス費用は平均3,000〜5,000円ほど。小さい金額だが、副業収入から引くと純利益のイメージが変わる。

このあたりを最初に知っていれば、もう少し計画的に動けたと思う。

---

## 4. AIが得意なことと、人間がやるべきこと

半年使って見えてきた「分担の法則」がある。

**AIが得意なこと**
- 構成案の大量生成
- 箇条書きや要点まとめ
- 文体のパターン変換（硬い→柔らかい、長い→短い）
- 類似表現の提案
- SEOキーワードの展開

**人間がやるべきこと**
- 読み手の感情を想像すること
- 一次情報や体験談の付加
- 最終的な「これで出せる」という判断
- クライアントとの関係構築

**AIは「考える道具」であって「考えてくれる存在」ではない。**

この感覚を早めに持てた人は、うまく使いこなせるようになると思う。

## 5. 実際に使ったAIツールと副業の種類

半年で試したAIツールは以下の7種類。副業との相性も含めて正直に書く。

### 使ったツール一覧と用途

| ツール | 主な用途 | 月額費用の目安 |
|---|---|---|
| ChatGPT（GPT-4o） | 記事構成・文章生成全般 | 約3,000円 |
| Claude（claude.ai） | 長文構成・ビジネス文書 | 約3,000円 |
| Perplexity | リサーチ・情報収集 | 約2,500円 |
| Notion AI | ドキュメント整理 | 追加なし（Notion内） |
| Canva AI | サムネ・図解制作 | 約1,500円 |
| Midjourney | アイキャッチ画像 | 約1,500円 |
| ElevenLabs | 音声コンテンツ | 使用頻度低・無料枠のみ |

全部使うと月1万円以上になる。実際には「よく使う3つ」に絞った方が効率がいい。自分のメインは ChatGPT＋Claude＋Perplexity の組み合わせで、これで副業収入の9割以上をカバーしている。

### 収入が増えた副業の種類と金額感

**① Webライティング（単価アップ交渉）**

AIで速度が上がった分、1案件あたりの調査・提案の質を上げることができた。結果として、既存クライアント2社から単価を1.2〜1.4倍に引き上げてもらえた。月+2万円。

**② 構成・ディレクション（新規開拓）**

「書かなくていい」仕事が取れるようになった。クライアントのざっくりした依頼をAIで構成に落とし込んで提案する。ライターを持っているクライアントに「ディレクターとして入る」形が取りやすくなった。月+2〜3万円。

**③ noteコンテンツ販売（自分発信）**

自分の体験と知識を記事にまとめて300円〜1,000円で販売。今は月5〜10本ほど売れている。月1〜2万円。まだ大きくはないが、積み上げている。

### 最も効率が良かった組み合わせ

Perplexity でリサーチ → ChatGPT で構成案を複数出す → Claude で文体を整える → 自分で体験談と感情を追加する。

この4ステップが今の基本ワークフローになっている。

## 6. 再現性のあるプロンプト設計の考え方

「プロンプトエンジニアリング」と聞くと難しそうに思えるが、実際に副業で使うレベルでは2つのことを意識するだけでかなり変わる。

### ①「役割」「条件」「アウトプット形式」の3点セット

プロンプトに必ず入れる要素：

```
役割：あなたは〇〇の専門家です
条件：読者は〇〇で、〇〇という課題を持っています
アウトプット：〇〇の形式で出力してください（例：H2見出し5つ、各200字）
```

これだけで出力の質が一段変わる。特に「読者の課題を明示する」のが効く。

**実際に使っているプロンプトの例（Webライティング用）**

```
あなたは月に30本以上の記事を書くプロのWebライターです。
以下の条件で記事の構成案を3パターン出してください。

テーマ：副業でAIを活用する方法
読者：月収10万円前後のフリーランサー。副業収入を増やしたいが、AIツールの選び方に迷っている。
目的：読者が「自分でも始められる」と感じること
構成：H2見出し5〜6つ、各200〜300字の概要付き
```

### ②「AI→人間→AI」の往復を使う

一発でいいものを出そうとするより、AIの出力に自分のコメントを加えて再度投げる方が精度が上がる。

具体的には：
1. AIに構成案を出させる
2. 「ここをこう変えたい」「この視点が足りない」と人間がフィードバック
3. 改善版をAIに出させる

このループを2〜3回まわすと、そのまま使えるレベルになることが多い。

### ③ 副業ジャンル別：よく使うプロンプトパターン

**ディレクション提案用（クライアント向け企画書）**

```
あなたは経験豊富なコンテンツディレクターです。
以下の情報をもとに、クライアントに提案する記事企画書を作成してください。

クライアントの業種：〇〇（例：中小企業向けのITツール販売）
ターゲット読者：〇〇（例：従業員10〜50人の中小企業の経営者）
クライアントの課題：〇〇（例：SEO流入が少ない、問い合わせが増えない）
アウトプット：記事タイトル案3つ、各タイトルの狙いキーワードと想定読者の悩みを200字で説明
```

このプロンプトを使うと、クライアントへの初回提案資料が30分以内に作れる。以前は2〜3時間かかっていた作業だ。

**noteコンテンツ販売用（有料記事の目次設計）**

```
あなたはnoteで月10本以上の有料記事を販売しているクリエイターです。
以下の条件で、300円で売れる有料記事の構成を提案してください。

テーマ：〇〇（自分の体験・専門知識を入れる）
読者の悩み：〇〇
無料部分：共感フック＋筆者紹介＋3つの気づき
有料部分：具体的な手順・ツール・数字ベースの実績
目次：6〜8項目、各項目の内容概要を100字で説明
```

プロンプト設計に悩む人は、まず「役割・条件・形式」の3点を意識するだけから始めてほしい。それだけでも出力のばらつきがかなり減る。

## まとめ

AIを副業に導入して半年、収入は月平均6〜7万円増えた。ただ「AIに全部任せて楽になった」というより「できることの幅が広がった」という感覚が正直なところだ。

最初の3ヶ月は誤算の連続だったし、今も失敗しながら試している。それでも、道具として使いこなすコツが見えてきてから、仕事の質と量が両方変わった。

**やっと「自分のペースで働いて、ちゃんと増える」という感覚が出てきた**のが正直なところだ。

月収10万円で停滞している人に伝えたいのは、「AIは魔法ではないけれど、使い方次第で確実に武器になる」ということ。
