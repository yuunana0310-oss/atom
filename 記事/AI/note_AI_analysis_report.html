04 — Engagement Analysis
コンテンツ類型別 エンゲージメント傾向 TOP 5
実収集データ（約90件分）のいいね数・コンテスト応募数・WebSearch補完情報から推定。 「代表いいね帯」は実観測値の中央〜上位域であり、確定的な平均値ではない。
  • 01
    個人体験×AI活用 転記・体験談型
    「分身AIを作ってみた（984いいね）」「1ヶ月ガチったら10万稼げた話」など。 note公式コンテストでも一貫して体験型が受賞。感情移入が起きやすくコメントも多い。 AIツールを使って「自分の何かが変わった」という構造がコンテスト受賞の共通パターン。
    500〜1,000
    ♡（代表帯）
  • 02
    AIツール実践解説型（非エンジニア向け）
    「限界まで実務でCursorを使ってみたら、アイアンマンの気分になれた（1,396いいね）」が観測最高値。 非エンジニア・ビジネスパーソン向けのCursor / Claude Code / NotebookLM解説が2025〜2026年トレンド。 Cursorワークショップ告知が427いいね。
    200〜1,400
    ♡（代表帯）
  • 03
    ランキング・比較レビュー型
    「個人的に触ったAIツール ベスト30」「おすすめAIツール 課金して比較（98いいね）」「AI執筆ツール10選」など。 有料記事の場合は1,000〜9,980円と幅広い。「本当に使い倒した」という一人称の正直レビューが差別化になる。
    100〜200
    ♡（代表帯）
  • 04
    AI副業・note収益化 ハウツー型
    「AI（ChatGPT）とnoteで月間3万PVのメディアを運営する方法（1,053いいね・有料9,980円）」が突出。 類似記事が乱立しているため差別化なしでは10〜50いいね止まりになりやすい。 実績数字と独自体験をセットにすることが必要と考えられる。
    20〜100
    ♡（代表帯）
  • 05
    AIニュースまとめ・日次更新型
    「ChatGPTニュースまとめ2026-03-25（38記事）」のような日次更新型。実観測いいね数は0〜4。 検索流入やシリーズ読者の維持には効くが、拡散型エンゲージメントは期待薄。
    0〜10
    ♡（代表帯）
05 — Title Patterns
AIカテゴリ 売れるタイトルパターン 6選
Pattern A — 体験×変化型
「〇〇を1ヶ月ガチったら、〇〇になった話」
「AIには書けない文章って、何？への回答」
読者の疑問に直接応答しながら書き手の体験を起点にする。「答え」を見せつつ体験談を語る構造がコンテスト受賞の共通パターン。
Pattern B — 【 】ブラケット＋具体ツール名
「【Kling3.0登場】AI動画が"ヤバすぎる領域"に突入」
「【2025年版】個人的に触ったAIツール ベスト30」
ツール名やバージョンで検索性と旬感が両立。「個人的に」という一人称が信頼感につながる。
Pattern C — 疑問→検証→発見型
「NotebookLMって何が凄いの？と疑問だったので使ってみた（151♡）」
「今、動画生成AIツールに課金するならおすすめは？への回答」
「疑問→検証→答え」の構造が読者の検索意図と一致しやすい。正直な疑問から始まるタイトルが共感を生む。
Pattern D — 数字×実績型
「AI×noteで月間3万PVのメディアを運営する方法（1,053♡・¥9,980）」
「限界まで実務でCursorを使ってみたら（1,396♡）」
具体的なKPIがタイトルに入ると、有料記事でも購買につながる。「月間3万PV」「1ヶ月10万円」が説得力を持つ。
Pattern E — 家族・身近な人×AI型
「80代母の『AIは不安』が変わった日（コンテスト受賞）」
「トラになりたい娘にとことん付き合う（コンテスト受賞）」
身近な人がAIと出会う体験記は世代を超えた共感を生む。2025年コンテストで一貫して見られるパターン。
Pattern F — 本音・立ち止まり型
「正直レビュー｜noteの相棒はどれ？AIを1ヶ月使い倒して出た結論」
「『SNS発信』が上手くいかない…とお悩みの方へ（230♡）」
「忖度なし」「本音」「正直に言う」フレームが信頼を得る。AI礼賛記事が飽和した2026年では差別化になると考えられる。
06 — Pricing Strategy
価格戦略と有料/無料の判断基準
コンテンツタイプ 推奨価格帯 根拠・コメント
個人体験談・エッセイ 無料（0円） コンテスト受賞作・高いいね記事のほぼすべてが無料。拡散力優先で認知を獲得する
プロンプト集・テンプレート 300〜980円 再利用性の高い実用資産。「すぐ使える」という即時価値で購買率が上がる
ツール活用ガイド・解説記事 500〜1,500円 実用系1,800円・読み物系980円が売れる目安（note公式データ参照）
ノウハウ体系・ロードマップ 1,000〜5,000円 「月間3万PV運営方法（9,980円・1,053♡）」のように実績×体系化で高単価が成立する
AIニュースまとめ・定期更新 無料またはサブスク低価格 いいね0〜4と拡散力が低い。継続読者獲得目的に限定する

売れるテーマ案 10選

  • 02

  • 03

  • 04
    「フリーランス2年目、AIを仕事に組み込んで変わったこととそのまま変わらなかったこと」

09 — Key Insight
AIカテゴリの勝利の方程式
Formula
個人の体験（失敗・発見・変化）
＋ 具体的なAIツール名と使い方
＋ 「正直に書く」スタンス
＋ 【 】タイトル or 旬のツール名・季節フック
なぜこれが効くのか
2025年のnoteコンテスト（Google Gemini×note）は計5回、応募総数56,000件超を集めた。 受賞作の共通パターンは「AIを使って自分の何かが変わった」という個人体験の記録。 ランキング・比較系・ニュースまとめ系は再現性が高い代わりに乱立して埋没しやすい。 月収10万前後のスキルワーカー向けには「自分も同じ状況だ」と感じさせる属性の明示が刺さる。 AI礼賛記事が飽和した2026年時点では、「うまくいかなかった」「立ち止まってみた」という誠実な視点が差別化になると考えられる。
