# 【月30分かかる業務資料が5分で完成】医療職が Claude Code で時間を取り戻した話

毎日30分。月6時間。年間72時間。―― 医療職のあなたが「業務資料作成」に費やしている時間が、実は回収可能だとしたら？

---

理学療法士として22年間、患者さんの治療と並行して「診療ガイド」「リハビリプログラム」「学習資料」といった業務資料を手書き・手入力で作り続けてきました。つい3ヶ月前までは「これは医療職の宿命」だと思っていました。でも、Claude Code を導入した3ヶ月間で、その時間を劇的に短縮できることに気づきました。

この記事では、「30分かかっていた業務が5分で完成した」という現実を、ありのままお伝えします。

## 1. なぜ30分もかかっていたのか

毎回の業務資料作成では、以下のステップを踏んでいました：

1. **コンセプト決定**（5分）：「今月の学習テーマは何か」を決める
2. **情報収集**（10分）：ガイドラインや論文から関連情報を探す
3. **構成・執筆**（10分）：Word か手書きで資料をまとめる
4. **校正・修正**（5分）：誤字脱字・医学的誤りをチェック

これが**毎月30分 × 12ヶ月 = 年間6時間**になっていました。

「医療職は忙しいから、これくらいは仕方ない」――そう思っていたのが、大きな勘違いでした。

---

## 2. Claude Code を導入した3つの理由

### 理由① 「手入力」という無駄に気づいた

あるとき患者さんから「先生、この資料って毎回手書きですか？」と聞かれました。その瞬間、「あ、同じ内容を毎回手で書き直してる」という非効率さに気づいたんです。

「なぜテンプレート化しないのか」と自問自答。答えは「テンプレート作成が面倒だから」でした。

### 理由② AIの進化を過小評価していた

「AI は医療には使えない」――そう漠然と思っていました。でも調べてみたら、Claude Code なら「医学的根拠を含めた資料生成」ができることが分かりました。

### 理由③ 時間は医療職の最大資源だから

患者さんと向き合う時間が増える。勉強時間が増える。副業（note執筆）の時間も増える。

「月30分」という小さな時間が、積み重なると大きな差になる実感を持ったから、導入を決めました。

---

## 3. 実際の変化（3ヶ月のタイムライン）

### 導入1週目：試行錯誤の段階

正直なところ、最初は失敗ばかり。「膝OA患者向けのリハビリプログラム表を作って」と指示しても、AIが生成したものは「医学的に正確だが、患者さんには難しすぎる」という状態でした。

**時間：30分のまま**（むしろ修正に時間がかかった）

### 導入2週目～4週目：「指示の工夫」に気づく

「強度レベル1～4で、関節可動域制限患者向けに」というように、より具体的に指示するようになりました。

そうすると、Claude Code の精度が劇的に上がった。

**時間：25分 → 20分に短縮**

### 導入1ヶ月～2ヶ月：仕組み化の段階

「よく使う指示」をテンプレート化し始めました。「膝OA患者向け」「肩関節痛患者向け」といった疾患別のプロンプト雛形を作ると、さらに効率化。

**時間：20分 → 10分に短縮**

### 導入3ヶ月目：定常状態

今では、業務資料の作成は以下のステップ：

1. 疾患名と対象患者を指定（1分）
2. プリセット済みプロンプトを実行（1分）
3. 出力内容をチェック（2分）
4. 修正指示（1分）

**時間：30分 → 5分に短縮 ✓**

---

## 4. 思わぬメリット・課題

### メリット① 資料の質が上がった

手書きの時代は「時間内にまとめる」ことが目標でした。でも今は「より医学的に正確で、患者さんにわかりやすい資料」を目指せるようになりました。

AI が「医学的根拠」を自動で補強してくれるから、私の負担が減って、質が上がる。これは予想外でした。

### メリット② 患者さんへの説明時間が増えた

浮いた25分を、患者さんへの個別相談に充てるようになりました。

「この運動が効く理由」を丁寧に説明できるようになり、患者さんの満足度も上がったように感じます。

### メリット③ 副業（note記事）の執筆時間が増えた

月30分の自動化で、note 記事執筆に30分プラスできました。副業の月収が確実に増えた実感があります。

### 課題① 「100%自動」ではない

AI が生成した資料が「完璧」なことはまずありません。医学的に正確でも、患者さんの理解度に合わない場合があります。

だから「AI が7割作って、私が3割修正」くらいの気構えが必要です。

### 課題② 指示が曖昧だと時間が増える

「リハビリプログラム作って」だけでは、AIが何を作ればいいか分からず、修正が増えます。

最初は「曖昧な指示を正確に変える」という追加作業が発生するので、その期間は時間短縮効果が薄れます。

## まとめ

「月30分の業務資料作成」という小さな時間が、3ヶ月で「5分」に短縮された。それは単なる時間短縮ではなく、**患者さんのためにする時間と、自分のキャリアのための時間が増えた**ということです。

医療職の「当たり前の業務」は、実は自動化の余地がたくさんあります。
