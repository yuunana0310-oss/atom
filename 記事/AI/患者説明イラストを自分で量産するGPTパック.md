## 1. 素材サイトを使い続けて気づいた「探す時間」の正体

forPTやイラストACのような素材サイトには、本当に助けられてきました。
無料で質の高い医療イラストが手に入るのは、現場にとってありがたい話です。

ただ、使い込むほどに気づくことがありました。
「探している時間」と「妥協している瞬間」が、地味に積み重なっているんです。

- 部位は合っているけど、強調したい箇所が違う
- 高齢の患者さんに渡すには、線が細くて見えにくい
- 説明したい角度（側面・後面）の図が見つからない
- 似た図はあるけど、自分の説明の流れに合わない

素材は「すでに完成しているもの」なので、こちらが図に合わせるしかない。
だから探す。見つからない。妥協する。あるいは手描きで足す。
この小さな摩擦が、1日に何度も起きていました。

AIを使い始めて変わったのは、ここでした。
**図を探すのではなく、説明したい内容を伝えて、図のほうを寄せてくる。**
順序が逆になっただけで、作業の質がまるで変わったんです。

## 2. forPTとの違い：「ある中から選ぶ」か「その患者向けに作る」か

誤解してほしくないのですが、forPTを否定したいわけではありません。
役割がそもそも違う、という話です。

| | forPT等の素材サイト | このGPTパック |
|---|---|---|
| 入手方法 | 完成済みの図から選ぶ | 説明したい内容から生成する |
| カスタム性 | ほぼ固定（編集は別ソフト） | 部位・角度・強調点・線の太さを指定可 |
| 対象者調整 | 図は1種類 | 高齢者向け・子ども向けなど作り分け |
| 探す時間 | 毎回かかる | 「作る」ので探さない |
| 著作権・利用規約 | サイトごとに確認が必要 | 生成物として自分の資料に使いやすい |
| 強み | 質の高い完成品がすぐ手に入る | その患者さん専用に寄せられる |

素材サイトは「質の高い完成品の在庫」。
このGPTパックは「その場で図を起こす設計図」です。

だから、両方使えばいいと思っています。
汎用的な図は素材サイトで十分。
でも「この患者さんに、この角度で、これを強調して伝えたい」が出てきた瞬間、素材は急に頼りなくなる。
そこを埋めるのが、このパックの役割です。

## 3. GPTパックでできること（用途別）

このパックは単なるプロンプト集ではありません。
**ヒアリング → 用途・対象の判定 → 情報密度の調整 → 失敗しにくい出力**までを1つの流れに組み込んだ、設計済みのGPTです。

| 使う場面 | 生成できるもの |
|---|---|
| 外来での口頭説明 | その場で見せる1枚の部位図・動作図 |
| 自主トレ指導 | 運動の手順イラスト（高齢者向けに大きめ・シンプル） |
| 家族への退院指導 | 在宅ケアの注意点を図解した1枚 |
| 院内掲示・待合 | 安心感のある説明ポスター構図 |
| note・SNS発信 | 目を引く医療ビジュアルの下絵 |

医療安全の配慮も設計に入れています。
不安を強く煽る表現、効果を保証する見せ方、過度なBefore/Afterは出にくいように組んであります。
心臓・解剖図など正確性が問われる領域は、断定を避ける言い回しを優先する設計です。

ここまでが無料パートです。
ここから先で、実際にGPTへ入れているヒアリング設計とプロンプト全文を公開します。
「コピペして自分のGPTに組み込めば、今日から同じ流れで量産できる」状態でお渡しします。

## 4. 【有料】実際のヒアリング設計とプロンプト全文

ここからが、このパックの核心です。
きれいな出力の裏にある「失敗しない順序」を、そのまま渡します。

### 4-1. GPT本体に入れる設計プロンプト（全文・コピペ可）

```
あなたは医療・リハビリ特化の患者説明イラスト設計アシスタントです。
ユーザー（医療職）が患者・家族に渡す説明イラストを生成します。

# 必ず守る順序
1. 目的を確認（口頭説明/自主トレ/家族指導/掲示/SNS）
2. 対象を確認（患者本人/家族/高齢者/子ども/一般）
3. 主役を1つに絞る（部位・動作・生活場面のいずれか1つ）
4. 情報密度を対象に合わせる（高齢者=線太め・要素少なめ）
5. 強調点を1つ確認してから生成する

# 出力ルール
- 主役は中央に大きく、補助要素は小さく
- 日本語の長文は崩れやすいので、ラベルは短い単語に
- 不安を煽る表現・効果保証・断定的な改善表現を使わない
- 解剖の正確性が必要な部位は「一般的には」等の表現を併記

まず目的だけを質問してください。一度に全部聞かないこと。
```

### 4-2. 用途別の追い込みプロンプト（5パターン）

実際に使っている、生成精度を上げるための指示文です。

```
【自主トレ図】
高齢患者向けの自主トレ手順イラスト。
動作は3ステップに分割、各ステップを横並び、番号付き。
線は太め、人物はシンプルな線画、背景は白。
強調したい関節を青で示す。文字は最小限。
```

```
【部位説明図】
（部位名）の側面図。患者本人への口頭説明用。
強調したい部位だけ色を変え、他は淡いグレー。
矢印やラベルは短い日本語1〜2語のみ。
```

```
【家族指導図】
在宅での（動作名）介助の注意点。
やりやすい姿勢／負担がかかりやすい姿勢の比較。
結果保証に見えないよう、改善を断定しない構図。
```

### 4-3. 失敗したときの立て直し手順

AI画像でいちばん心が折れるのが「混入」と「崩れ」です。
ここを乗り越える手順も入れています。

- 別案件の要素が混ざったら → 「何が混入したか」を先に言語化してから再生成
- 文字が崩れたら → ラベルを画像生成に任せず、後からCanva/PowerPointで載せる構図に切り替え
- 解剖が破綻したら → 角度を変える指示を足す（「真側面から」など固定語を使う）

この3つを知っているだけで、再生成のループから抜けやすくなります。

### 4-4. 自分の科・専門に寄せるカスタマイズ

整形・脳神経・呼吸器など、科ごとに頻出部位は違います。
設計プロンプトの「主役候補」に自分の科の頻出部位を3〜5個書き足すだけで、
ヒアリングが一段早くなります。具体的な書き足し例も同梱しています。
