# 【Claude Codeって何？】非エンジニア医療職が5分で理解するAIコード自動生成ツール

「プログラミングなんて自分には無理」――医療職のあなたが何度も思ってきたその言葉が、実は自分の可能性を無意識に制限していたことに気づく瞬間が、この記事にあります。

---

## 筆者の背景

理学療法士として22年間の臨床経験がある私も、つい3年前までは「コードなんて読めない。自分はAIツールの使い手にはなれない」と勝手に制限していました。患者記録に毎日30分、計画書作成に1時間。その時間をなんとかしたいという一心で、AI時代の医療現場に飛び込んでみた結果、Claude Codeというツールに出会いました。

**実際に実装した結果**、かつて手書きで1時間かかっていた業務が10分に短縮されました。その浮いた時間を患者さんのリハビリプログラム開発に充てられるようになり、診療の質も確実に上がった実感があります。

## 1. Claude Codeってそもそも何？

**簡潔に言うと：「AIがあなたの代わりに小さなプログラムを書いてくれるツール」**です。

もう少し詳しく：
- あなたが「〇〇したい」と日本語で指示する
- Claude Code（AI）がその指示を読んで、自動的にコードを書く
- 書かれたコードを実行すると、望みの結果が得られる

**重要なポイント**：
あなたはコードを理解したり、コードを書いたりする必要がまったくありません。「こういう結果が欲しい」という**「何がしたいか」を言葉で説明するだけ**で、AIが勝手に仕事をしてくれます。

## 2. ChatGPT・Claudeアプリとの違い

ここで混乱しやすいので、整理しておきます。

| 比較項目 | ChatGPT | Claude（通常アプリ） | **Claude Code Web版** |
|---|---|---|---|
| **アクセス方法** | ブラウザまたはアプリ | ブラウザ | **ブラウザ（`claude.ai/code`）** |
| **できること** | 文章作成、質問回答 | 文章作成、質問回答、分析 | **プログラム自動生成＋実行** |
| **あなたの作業** | 回答をコピペする | 回答をコピペする | ほぼなし（実行ボタンを押すだけ） |
| **時間短縮** | 中程度 | 中程度 | **大きい** |
| **医療職向け** | △（文章作成向け） | △（分析・説明向け） | **◎（業務自動化向け）** |

**注記**：Claude Code には Web版の他に CLI（コマンドライン）や IDE拡張がありますが、この記事では医療職が最もアクセスしやすい **Web版（ブラウザ）** に絞って解説します。

## 3. 医療職でも使える理由

**「プログラミングができない」という条件下で、Claude Codeが仕事をしてくれる理由：**

### 理由① AIが「意図を理解する」力が高い
あなたが「患者のリハビリプログラムを週ごとの進度表に変換したい」と日本語で言えば、AIはその意図を読み取り、必要なコードを自動生成します。プログラミング言語の文法を知る必要はゼロです。

### 理由② 実行環境がブラウザに組み込まれている
Claude Codeはブラウザ上で完結します。サーバーをセットアップしたり、複雑な環境構築をしたりする必要がありません。

### 理由③ 医療職が必要な作業は「単純で繰り返し」が多い
- 学習資料・論文の整理
- 診療ガイドラインの整理
- テンプレートへの自動入力
- リスト→表への変換

これらの業務は、プログラミングの知識がなくても指示できるものばかりです。

## 4. 医療現場で実装できる具体例

実際に、**あなた個人が**この月から実装できることを4つ挙げます。組織全体ではなく、自分の時間が減る・自分のスキルが上がることばかり。

### 例① 疾患別の学習ノート自動作成
**Before**: 学会発表や論文で学んだことを手書きノートに整理、1時間かかる

**After**（私の実装では）: 論文テキストやメモを貼り付けて「膝OAの運動療法のポイントを5行でまとめて」と指示 → 5分で完成

**何が起きるか**: Claude Codeが自動的に疾患の要点をまとめた学習ノートを生成。自分の疾患知識が体系化される。

---

### 例② 自分用のリハビリプログラム作成
**Before**: 指導用の運動プログラムを毎回手書きで作成、15分

**After**（私の実装では）: 「60代の膝OA患者向けに4週間の自宅運動プログラムを作って」と指示 → 2分で完成

**何が起きるか**: 強度や難易度が段階的に上がるプログラムが自動生成。自分の指導の引き出しが増える。

---

### 例③ note・ブログ記事の自動作成
**Before**: 自分が学んだ医療知識をnote記事にするのに1時間かける

**After**（私の実装では）: 学習メモを貼り付けて「医療職向けのnote記事にして」と指示 → 15分で下書き完成

**何が起きるか**: Claude Codeが記事構成を整えてくれる。修正するだけで記事が完成。副業の記事量産が加速する。

---

### 例④ 自分の学習ログの自動整理
**Before**: 毎日学んだ疾患知識・手技のポイントを手書きログで記録、1日10分

**After**（私の実装では）: 学習メモを音声入力→「学習ログにまとめて」と指示 → 2分で完成

**何が起きるか**: Claude Codeが自動的に学習内容を整理・体系化。毎日の経験が知識資産になる。

## 変わらなかったこと・正直な感想

ここまで「すごい！」という話ばかり書きました。ただ、正直に言うと：

- **完璧ではない**：AIが生成したプログラムが100%希望通りになるわけではありません。「ちょっと直して」という修正が1〜2回入ることもあります。でも、その修正を含めても手書きより圧倒的に早い。

- **指示の工夫が必要**：「患者記録を整理して」よりも「患者の年齢・診断名・治療期間を別々の列にして、改善率の高い順に並べて」という具体的な指示の方が、精度が上がります。ただ、これも医療職なら簡単にできる工夫です。

- **医療の「判断」は人間の仕事**：AIが自動化できるのは「定型業務」だけです。患者さんの治療方針を決めたり、異常に気づいたりという「医療職としての思考」は、人間にしかできません。むしろAIに業務を任せることで、その高度な思考に時間を使えるようになります。

Claude Code は、医療職の「毎日30分かかる業務」を「5分」に短縮する現実的なツールです。
