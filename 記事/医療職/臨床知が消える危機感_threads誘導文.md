## ① 共感フック型（本命・168♡型に最も近い）
```
後輩に「なんでまだ見てもないのにわかるんですか？」と聞かれて、
うまく答えられませんでした。

「なんとなく、今までの患者さんと似てるから」としか言えない。
でもその"なんとなく"には、22年分の患者さんの顔が詰まってるはずなんです。

この"なんとなく"、私が辞めたら消えるんだなと気づいて、ちょっと怖くなった話を書きました。
```

## ② 問いかけ型（保存・コメントを誘う）
```
医療職のいちばんおいしい部分って、
教科書じゃなくて「その人にしかない見方」だと思うんです。

・この人は家で頑張りすぎるタイプだ
・この表情は痛みより不安が大きい
・この歩き方、来週崩れるかも

これ、カルテにも書かれないまま消えていきますよね。
もったいなくないですか？
```

## ③ 自己開示・弱み型（れんとんの"無職"型）
```
知識はあるのに、形にできない。

ノートに書こうとしても続かない。
勉強会資料を作ろうとしても、見た目を整えてるうちに夜が更けて力尽きる。

正直、悔しかったです。
「伝えたいことは山ほどあるのに」って。

同じモヤモヤ、抱えてる医療職の人いませんか。
```

## ④ 気づき・転換型
```
「教科書みたいにきれいにまとめよう」をやめたら、
はじめて手が止まらなくなりました。

完璧に残そうとしない。
頭の中をまず一回、外に出してみる。

それだけで、22年分の"なんとなく"が言葉になり始めたんです。
```

## ⑤ ストーリー断片型（記事の山場を切り出し）
```
退院間際、患者さんは「もう大丈夫です」と笑ってた。
でも荷物のまとめ方が、やけに急いでた。

長く見てると、この「言葉と動作のズレ」が気になるんです。
結局その方、家に帰る不安を言えずにいただけでした。

この"気になった理由"を後輩に説明できなくて——という話を書きました。
```
