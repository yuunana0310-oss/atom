# 🎨 Visual Creative Studio

**対話しながらビジュアルコンテンツを作れるWebアプリ**

Instagram向け図解カルーセル、YouTubeサムネイル、4コマ漫画、ストーリー漫画、
サービス紹介ビジュアル、固定キャラクター展開、世界観統一クリエイティブ一式を、
チャット形式の対話で作成できます。

## 特徴

- 💬 **チャット形式の対話** — 作りたいものを自然な言葉で伝えるだけ
- 🎯 **段階的な制作** — 足りない情報だけ順番に確認、情報が十分ならすぐ生成
- ✏️ **修正指示に対応** — 「もっと高級感」「2枚目だけ変えて」などの差分更新
- 👤 **固定キャラクター** — キャラ設定を保持したまま複数枚展開
- 🌍 **世界観統一** — 配色・トーン・ジャンル感を揃えた複数クリエイティブ一括生成
- 📱 **5つの制作モード** — デザイン / 4コマ漫画 / 1ページ漫画 / キャラ展開 / 統一セット
- 🔧 **外部API不要** — モックモードで完全に動作（API設定時はLLM連携可能）
- 💾 **SVG/HTML/JSON出力** — ダウンロード＆再利用可能

## セットアップ

### 必要環境
- Python 3.10+

### インストール

```bash
cd visual-creative-studio
pip install -r requirements.txt
```

### 起動

```bash
python app.py
```

ブラウザで http://127.0.0.1:5050 を開いてください。

## 使い方

### 基本的な流れ

1. チャットに作りたいものを入力
2. AIアシスタントが不足情報を質問
3. 回答すると自動で生成
4. プレビューで確認
5. 修正指示で調整
6. SVG/JSONでエクスポート

### 入力例

| やりたいこと | 入力例 |
|-------------|--------|
| Instagram図解カルーセル | `Instagram用に図解カルーセルを作りたい。AI活用の基本を5枚で` |
| YouTubeサムネイル | `YouTubeサムネで、初心者向けClaude Code入門っぽく` |
| サービス紹介 | `サービス紹介画像を親しみやすく作って` |
| 4コマ漫画 | `4コマ漫画で、AIを使いたいけど最初の一歩が怖い人あるある` |
| 1ページ漫画 | `プログラミング初心者の「あるある」を1ページ漫画5コマで` |
| 固定キャラ展開 | `このキャラを固定したまま3枚展開して。名前はアイちゃん` |
| 世界観統一セット | `世界観を統一して、サムネ・カルーセル・紹介画像を一式作って` |
| 修正指示 | `もっと高級感を出して` / `2枚目だけ文字を減らして` |

### デザインスタイル

| スタイル | 特徴 |
|---------|------|
| シンプル | 洗練されたミニマルデザイン |
| ポップ | 明るく楽しい雰囲気 |
| 高級感 | 重厚で上品、ゴールドアクセント |
| ビジネス | 信頼感のあるプロフェッショナル |
| 親しみやすい | 温かみのあるフレンドリーデザイン |

### サイズ対応

- Instagram ポートレート: 1080 × 1350
- Instagram ストーリー: 1080 × 1920
- YouTube サムネイル: 1280 × 720
- note / OGP: 1200 × 630
- 4コマ漫画: 800 × 1200
- 1ページ漫画: 800 × 1100

## UI構成

- **左サイドバー**: セッション一覧、新規作成
- **中央チャット**: 対話エリア、サンプル入力、キャラ設定
- **右プレビュー**: SVGリアルタイムプレビュー、スライド切替、エクスポート

## 技術構成

```
Flask (Python) + Vanilla HTML/CSS/JS
├── core/          # 意図解析・会話エンジン・状態管理
├── adapters/      # LLMアダプタ (Mock / OpenAI)
├── generators/    # SVG生成 (デザイン/漫画/キャラ/統一セット)
├── templates/     # HTMLテンプレート
├── static/        # CSS/JS
└── data/          # セッション・履歴保存
```

## LLM連携（オプション）

`.env` に OpenAI API キーを設定すると、LLM連携モードに切り替わります。

```
OPENAI_API_KEY=sk-your-key-here
```

未設定の場合はモックモード（ルールベース）で動作します。
