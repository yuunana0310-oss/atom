# 🎯 全社メインテーマ戦略方針書

**決定日：2026年3月29日**
**決定者：社長（ユーザー）**
**適用範囲：全部門・全エージェント**

---

## ■ メインテーマ（最重要・全員遵守）

> ### **「AI×医療（医療職×AI副業）」**

「理学療法士がAIを使って副業・プロダクト開発・業務効率化を実現するまで」を軸に、全発信・全コンテンツを統一する。

---

## ■ なぜこのテーマか（決定背景）

| 軸 | 根拠 |
|---|---|
| 競合優位性 | 「AI×医療職の副業」発信者は日本にほぼ存在しない（ブルーオーシャン） |
| 参入障壁 | 理学療法士の国家資格が信頼を自動担保。誰でも真似できない |
| 市場規模 | 医師・看護師・PT・OT・ST等 医療職は日本に約400万人 |
| 収益性 | 医療職は購買力が高く、コンサル単価も30〜100万円以上を狙える |
| 相乗効果 | リハカルテ開発という実績が「体験談」として最強コンテンツになる |

---

## ■ ターゲットペルソナ

**Primary（メイン）：** 医療職（PT・OT・ST・看護師・医師）でAIに興味があるが怖い・難しいと感じている人。副業・業務効率化・現場改善をしたいと思っている。

**Secondary（サブ）：** 非エンジニア全般で「専門職×AI」の可能性に共感できる人。

---

## ■ 発信の核となる3つのストーリー

1. **現役PTがClaude Codeでリハカルテを自作した話**（実績の証明）
2. **医療現場でAIを使って業務がどう変わったか**（共感・体験談）
3. **医療職が安全にAI副業を始める方法**（有料コンテンツの入口）

---

## ■ 収益ロードマップ

```
Phase 1（〜3ヶ月）：信頼構築期
  → 体験談投稿でフォロワー獲得
  → note無料記事で集客

Phase 2（3〜6ヶ月）：収益化開始
  → note有料記事（980〜1,980円）：医療職向けAI入門
  → note有料記事（3,000〜5,000円）：リハカルテ開発全手順

Phase 3（6ヶ月〜）：高単価化
  → 医療職向けAI副業コンサル（30,000〜100,000円/人）
  → 医療法人・クリニック向けAI導入支援（月額顧問）
```

---

## ■ 全部門への指示

| 部門 | 対応内容 |
|---|---|
| SNS・マーケティング部 | 投稿テーマを「AI×医療職」に統一。他ジャンルは原則発信しない |
| Threads運用部 | account_profile・theme_treeをAI×医療軸に更新済み（参照） |
| コンテンツ制作 | 記事テーマは医療職×AI副業に絞る。汎用AI情報は控える |
| リハカルテ製作部 | 開発ストーリーをコンテンツ素材として積極的に記録・共有する |
| リサーチ・企画部 | 医療職×AI市場の動向・競合調査を優先的に実施する |
| 販売戦略部 | 医療職向けの価格帯・販売チャネルを設計する |

---

## ■ やってはいけないこと

- テーマをブラさない：「AI×副業（汎用）」や「Threads自動化」に軸を変えない
- 医療情報の無断断言：医療的な効果・効能の断言は薬機法・景表法上のリスクがある
- 競合の誹謗中傷：他の医療系発信者を貶める比較は厳禁
