## お願いの背景

高齢者の誤嚥リスクが高い患者に対して、STが不在または介入が手薄な状況でも**PT・OTが関わられる嚥下訓練の選択肢を整理したい**と考えました。
医学論文・ガイドラインをもとにAIを用いてまとめたレポートを2本作成しましたので、内容の妥当性についてご確認いただけますでしょうか。

## 提出資料

### ① `swallowing_pt_evidence_report.html`
**PT・OTが施行可能な嚥下関連訓練 全体レポート（13訓練）**

- PT訓練 7種：Shaker Exercise / EMST / 体幹・姿勢訓練 / 廃用予防・離床促進 / 頸部可動域訓練 / バランス訓練 / 呼吸訓練
- OT訓練 6種：口腔機能訓練 / 摂食動作・自助具 / 認知機能介入 / 食事環境調整 / 口腔ケア支援 / 上肢巧緻性訓練
- 各訓練に実施プロトコル・引用文献・業務範囲上の根拠を記載

### ② `shaker_exercise_evidence_review.html`
**Shaker Exercise（頭部挙上訓練）一次文献レビュー**

- 主要5論文を横断的に整理（対象・n数・結果・バイアスリスク）
- 適応・禁忌・日本の高齢入院患者への適用限界を明示
- 代替手技CTAR（Chin Tuck Against Resistance）との比較を記載

## 特にご確認いただきたい点

1. **訓練の適応・禁忌の記載に漏れや誤りがないか**
2. **引用論文の解釈に臨床的に問題がないか**（特にShaker Exerciseの適応患者像）
3. **日本の病棟・施設環境でPT・OTが単独実施できる訓練の範囲として妥当か**
4. **追加すべき訓練・削除すべき訓練があるか**

## レポートの限界（事前にご共有）

- エビデンスレベル（RCT/SR等のバッジ）は**論文の研究デザインをもとにAIが独自分類したもの**です。GRADEなどの公式評価システムには準拠しておらず、研究の質（バイアスリスク・サンプルサイズ等）の評価は未実施です。
- 引用論文はAIが抄録から判断したものであり、**一部に誤引用が確認されています**（Shaker 1997の掲載誌の誤記）。全文確認は未実施です。
- あくまで**訓練の選択肢と概要を把握するための叩き台**として作成しています。
