# 🚫 やっちゃいけないこと（atom）

## 絶対NGワード（投稿文に入れない）
| ワード | 理由 |
|---|---|
| 絶対 | 断言が強すぎる・責任を持てない |
| 必ず稼げる | 景表法・誇大表現 |
| 楽して | 安易さを煽る |
| 誰でもできる | 読者をなめている |
| 簡単に稼げる | 誇大表現・信頼失墜 |
| 不労所得 | ジャンルのイメージが悪い |
| リスクゼロ | 嘘になる |
| 100%・保証 | 根拠のない断言 |
| 詐欺・違法・グレー | 関わってはいけない領域 |
| ネットワークビジネス・MLM・マルチ | アカウントの信頼を壊す |
| 副業詐欺・情報商材 | これ自体が悪評を招くワード |

## 要注意ワード（使う場合は根拠を添える）
- 最高の / 最強の / 唯一の / 完璧な → 根拠なく使わない

## コンテンツとして書いてはいけないこと

### 医療・健康情報
―病院・施設名は出さない
- 根拠のない医療情報・健康効果の断言
- 「〇〇で病気が治る」「〇〇をやめたら完治した」系
- 特定の治療法・医薬品の推奨（医師でも判断が難しい領域）

### 法律・規制
- 労働法・税法に違反する副業手法の紹介
- 著作権・肖像権を侵害するコンテンツ
- 個人情報（患者・職場・知人）を特定できる内容
- 医師法に抵触する内容


### 競合・他者
- 特定の個人・企業への批判・悪口
- 根拠のない比較優位（「〇〇より絶対いい」）

### アカウント戦略上NG
- **「患者の自主トレ」や「リハビリの手技」ばかりに偏った投稿（※多様な内容が枯渇し同じ話題ばかりになるのを防ぐため、AI活用や働き方など幅広くテーマを選ぶこと）**
- 政治・宗教・社会問題への明確な立場表明
- 炎上リスクの高いテーマ（意見が二分するトピック）
- フォロワーを煽る表現（「これを知らない人はやばい」等）

## 投稿スタイルのNG

### やっちゃいけない書き出し
- `今日はAIについて書きます` → 何を言いたいかわからない
- `みなさんこんにちは` → スクロールされる
- `〜について解説します` → 内容が見えない
- `実は〇〇なんですよね〜` → 軽すぎる

### やっちゃいけない構成
- 1投稿に複数のCTA（「noteも見て・フォローして・コメントして」）
- テーマを詰め込みすぎ（1投稿1テーマ厳守）
- 感情語の乱用（1投稿1〜2個まで）
- 280文字を超える長文（読まれない）

### 頻度・運用上のNG
- 1時間以内に連投（bot判定リスク）
- 1日10件超の投稿（同上）
- 過去100件以内に類似した内容を再投稿（重複・マンネリ）
- 同じテーマを2件連続で投稿

## 緊急停止が必要な状況
以下が発生した場合はSupervisorがKILL_SWITCHを発動する：
- API認証エラー（凍結リスク）
- 重複投稿が2件以上発生
- 1時間に10件超の投稿を検知
- ユーザーから停止命令
