# セットアップ手順

## 必要なもの一覧

| 項目 | 必須 | 用途 | なしの場合 |
|---|---|---|---|
| Anthropic APIキー | ✓ | 投稿文をClaudeで生成 | モック文章で動作 |
| Threadsアカウント | ✓ | 投稿するアカウント本体 | - |
| Threads アクセストークン | ✓ | 投稿・メトリクス取得 | モック動作（実投稿されない） |
| Threads ユーザーID | ✓ | API呼び出し時のユーザー識別 | 同上 |
| YouTube APIキー | 任意 | ネタ収集の自動化 | 組み込みモックデータで動作 |

---

## 1. Anthropic APIキー取得

**時間目安: 5分 / 難易度: ★☆☆**

1. https://console.anthropic.com/ にアクセス
2. Googleアカウントでサインアップ
3. 左メニュー「API Keys」→「Create Key」
4. キーをコピー（**一度しか表示されない**）

**料金**: 従量課金。デフォルト設定（Haiku）なら1,000投稿生成しても数百円レベル。

---

## 2. Threadsアカウント作成

**時間目安: 5分 / 難易度: ★☆☆**

1. スマホでThreadsアプリをDL
2. Instagramアカウントと連携して作成
   - Instagramアカウントがない場合は先に作成
3. プロフィールを設定しておく（API取得前に済ませる）

---

## 3. Threads APIアクセストークン取得

**時間目安: 30〜60分 / 難易度: ★★★**（唯一のハードル）

### Step1: Meta for Developersにアクセス
- https://developers.facebook.com/
- Facebookアカウントでログイン（なければ作成）

### Step2: アプリを新規作成
1. 「マイアプリ」→「アプリを作成」
2. タイプ: 「その他」→「消費者」を選択
3. アプリ名は何でもOK（例: `threads-bot`）

### Step3: Threads APIを製品として追加
1. アプリダッシュボード → 「製品を追加」
2. 「Threads API」を選択して追加

### Step4: テストユーザーを追加
1. 左メニュー「アプリロール」→「ロール」
2. 「Threadsテストユーザー」→ 自分のThreadsアカウントを追加

### Step5: アクセストークンを取得
1. 左メニュー「Threads API」→「APIを調査する」
2. 以下の3スコープにチェック:
   - `threads_basic`
   - `threads_content_publish`
   - `threads_manage_insights`
3. 「Generate Token」→ 表示されたトークンをコピー

### Step6: ユーザーIDを取得
ブラウザで以下にアクセス（`{TOKEN}`を取得したトークンに置き換え）:
```
https://graph.threads.net/v1.0/me?fields=id&access_token={TOKEN}
```
返ってくるJSONの `"id"` の値が `THREADS_USER_ID`

```json
{"id": "1234567890"}  ← この数字がUSER_ID
```

### ⚠️ トークンの期限
テスト用トークンは**60日で失効**する。本番運用が安定したら長期トークンへの変換が必要。

## 4. YouTube APIキー取得（任意）

**時間目安: 10分 / 難易度: ★☆☆**

1. https://console.cloud.google.com/ にアクセス
2. プロジェクトを新規作成
3. 「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
4. 「認証情報」→「認証情報を作成」→「APIキー」
5. キーをコピー

**料金**: 1日10,000クエリまで無料（このシステムでは余裕で収まる）

## 5. .envファイルに記入

`C:\Users\yuuna\agens\threads-agent\.env` を開いて記入:

```env
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxx
THREADS_ACCESS_TOKEN=THQWxxxxxxxxxxxxxxxxx
THREADS_USER_ID=1234567890
YOUTUBE_API_KEY=AIzaxxxxxxxxxxxxxxxxx
OPERATION_MODE=review
```

`OPERATION_MODE`:
- `review` = 5%人間モード（推奨）
- `auto` = 完全自動
