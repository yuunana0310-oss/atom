## 【法的根拠】

### ✅ 常勤の定義（介護保険法）

#### 出典
- 厚生労働省老健局「人員配置基準等」（2021年4月1日）
- 社会保障審議会 介護給付費分科会第223回資料2

#### 法定内容

```
「常勤とは、当該事業所において常勤の従業者が
 勤務すべき時間数に達している従業者をいう。

 ただし、常勤の従業者が勤務すべき時間数が
 週32時間を下回る場合は、週32時間を基本とする。」
```

#### 解釈ポイント

| ポイント | 内容 |
|---------|------|
| **雇用形態の無関係性** | 正社員・パート関係なく、勤務時間数で判定 |
| **施設の定義権** | 施設が「常勤 = 週○時間」と就業規則で定めることが可能 |
| **最低基準** | 週32時間未満に設定することは不可（自動修正される） |
| **上限基準** | 特に定められていない（通常は週40時間） |

#### 結論

**パートタイムでも、施設の定めた常勤基準に達していれば「常勤」として認定される**

---

### ✅ 個別機能訓練加算の常勤要件

#### 出典
- 厚生労働省「令和3年度介護報酬改定Q&A」
- 特別養護老人ホームの個別機能訓練加算（Ⅰ）・（Ⅱ）

#### 法定内容

| 加算区分 | 常勤要件 |
|---------|---------|
| **個別機能訓練加算（Ⅰ）イ** | 常勤専従の理学療法士等を配置 |
| **個別機能訓練加算（Ⅰ）ロ** | 常勤専従1名 + 常勤換算（入所者数÷100）以上 |
| **個別機能訓練加算（Ⅱ）** | 常勤換算で入所者30名につき1名以上 |

#### 対象資格

```
✅ あん摩マッサージ指圧師は認定資格
   （その他：理学療法士、作業療法士、言語聴覚士、
    看護職員、柔道整復師、鍼灸師）
```

---

### ✅ 時間帯分離による「専従」認定

#### 出典
- 厚生労働省「令和3年度介護報酬改定Q&A Vol.3」
- 「管理者と機能訓練指導員の兼務について」

#### 法定内容

```
「時間帯を分離することにより、
 各々の職務を『専従』として認定できる。

 例：管理者として3時間、
    機能訓練指導員として5時間の場合、
    
    5時間の部分は『機能訓練指導員専従』と認定」
```

#### 実務的意味

| 状況 | 認定 |
|------|------|
| 同時に両職務を兼務 | ✅ 不可（『専従』にならない） |
| 時間帯で完全分離 | ✅ 可能（各時間帯で『専従』と認定） |

---

### ✅ 看護職員と機能訓練指導員の兼務

#### 出典
- 厚生労働省「令和3年度介護報酬改定Q&A」
- 「看護職員と機能訓練指導員の兼務について」

#### 法定内容

```
「看護職員としての業務に従事していない時間帯において、
 機能訓練指導員として勤務することは差し支えない。
 
 この場合、その時間帯は『機能訓練指導員専従』と認定できる。」
```

---

### ✅ 複数人の常勤換算配置

#### 出典
- 特別養護老人ホームの人員配置基準
- 個別機能訓練加算の算定要件

#### 法定内容

```
入所者が100名を超える場合：

「常勤専従1名 ＋ 常勤換算（入所者数÷100）以上」

例）入所者150名の場合
常勤専従：1名（必須）
常勤換算：1.5名以上（追加）
合計：常勤換算2.5名分以上
```

#### 実務的意味

- 複数人を「常勤換算」で配置することは制度上認められている
- パートタイム職員も常勤換算に含まれることは周知の事実
