# デザインスタイル辞書

このファイルは、デザインスタイル番号から該当スタイルを参照するためのナレッジ用辞書です。

## 使い方の前提
- ユーザーが `No.37`、`37番`、`301で` のように番号指定した場合、該当する番号を優先参照する。
- 通常時は300件を毎回列挙せず、用途・目的・ターゲットに応じて3〜5案だけ提案する。
- 単体スタイルでは再現性が低い場合のみ、主スタイル1つ＋補助スタイル最大2つの複合型を提案してよい。
- この辞書は「スタイルの内容」を保持するためのものであり、運用ルールは別ファイルの「デザインスタイル検索ガイド.md」またはInstructionsを優先する。

## 項目の見方
- **No**: スタイル番号
- **区分**: 既存 / 非医療追加 / 変化球追加 / かわいい系追加 / 尖りかわいい追加 など
- **コンセプト**: そのスタイルの骨格
- **相性がいい用途**: 向いているテーマや媒体
- **印象**: 見た時の空気感やトーン
- **推奨カテゴリ**: 大まかな分類・検索補助
- **備考・ビジュアル要素**: 画面づくりのヒント
