## ⚠️ 最重要免責

- ACL（前十字靱帯）再建術後のリハは、**移植腱の種類・半月板処置の有無・軟骨損傷の有無**で進行が大きく変わる。
- 復帰は**時期だけでなく機能的基準**を満たすことが必須。「術後◯か月で復帰」を一律で示してはならない。
- 本書のフェーズ・期間・基準値は**目安**であり、医師指示・施設プロトコルが優先される。
- 心理的準備（ACL-RSI）も復帰判断に組み入れる。
- 過早復帰は**再断裂リスクを大きく上昇させる**ため、判断を急がない。

## 3. 共通している内容

### 3-1. 復帰は基準ベース（時期だけで決めない）
S1, S2, S4, S6で一致。**最低6か月**を一つの目安としつつ、**機能的基準達成**を必須とする。

### 3-2. LSI（Limb Symmetry Index）≥90%
- **大腿四頭筋強度**：等速性筋力測定でLSI≥90%
- **ホップテスト**：シングル、トリプル、クロスオーバー、6mタイマーホップで各LSI≥90%

### 3-3. 心理的準備（ACL-RSI）
復帰判断に**心理的準備度**を組み入れる（S2, S5）。閾値は研究で異なるが**60〜65点以上**を目安とすることが多い。

### 3-4. 動作の質（mechanics）も評価
LSIだけでは不十分。**着地時の膝外反・体幹コントロール・対称性**を観察する（S2, S4）。

### 3-5. 神経筋トレーニングの組み込み
バランス・プライオメトリクス・スポーツ特異動作を段階的に。

### 3-6. 再損傷リスクの認識
復帰後**最初の数か月で再断裂・反対側ACL損傷リスクが最も高い**。

## 4. 情報源ごとに違いがある内容

### 4-1. 復帰時期の目安
| ソース | 内容 |
|--------|------|
| S2 Aspetar 2023 | **9か月以前の復帰は推奨しない**。多くの場合12か月前後が安全 |
| S4 Grindem 2016 | 9か月以降＋基準達成で再損傷リスク84%減 |
| S6 日整会2019 | 単束再建：4〜6か月、二重束再建：6〜9か月が目安（基準達成前提）<br>※ Aspetar 2023（S2）はより保守的な立場で「9か月以前は推奨しない」。GPTはスポーツ復帰について、時期のみではなく、9か月以降を基本目安とし、筋力・ホップテスト・心理面・動作の質・医師許可を含めて総合判断する方針で回答する。 |
| 一部施設プロトコル（S8, S9） | 6か月以降を目安にしつつ基準ベース |

→ **GPTの方針**：「術後9か月以前の復帰は再損傷リスクが高い」「基準達成が前提」と回答。

### 4-2. 移植腱種類による差
| 移植腱 | 特徴 | 配慮事項 |
|--------|-----|---------|
| BPTB（骨付き膝蓋腱） | 骨-骨癒合で固定強い | **大腿四頭筋強度回復が遅れやすい**、膝前面痛・膝立ち困難 |
| ハムストリング腱（HT） | 採取部痛が少ない | **ハムストリング筋力低下が長期に残りやすい**、ハム強化重視 |
| 大腿四頭筋腱（QT） | 比較的新しい選択肢 | 早期の骨統合（4〜6週）あり、膝伸展機構の保護必要 |

→ **復帰の閾値（LSI≥90%等）は腱種類で変えない**ことが推奨されているが、**到達までの時間**が異なる。

### 4-3. 半月板処置を伴う場合
- 半月板**縫合**を併用：**深屈曲・荷重制限を術後4〜6週**設けることが多い（`06_半月板縫合術_術後リハ.md` 参照）
- 半月板**部分切除**：制限はACL単独に近い

### 4-4. 軟骨損傷を伴う場合
- マイクロフラクチャー・軟骨修復を併用：**荷重制限を6〜8週**設けることが多い
- 部位（荷重面か非荷重面か）でも変わる

### 4-5. ランニング開始時期
| ソース | 内容 |
|--------|------|
| 多くの施設 | 術後**12〜16週**でストレートランから開始（基準達成前提） |
| Aspetar | 直線走行は条件達成後、軽負荷から段階的に |
| 共通 | **大腿四頭筋LSI≥80%・痛みなし・ホップテスト準備状態**が前提 |

### 4-6. ジャンプ開始時期
| ソース | 内容 |
|--------|------|
| 多くの施設 | 術後**16〜20週**から両脚ジャンプ、その後片脚ジャンプ |
| 共通 | LSI≥85%、フォーム良好、痛み・腫脹なし |

## 5. GPT Knowledge本体

### 5-A. 概要

ACL再建術はスポーツ外傷で最も頻度が高い靱帯再建のひとつ。再建後のリハビリは**12か月以上の長期戦**であり、急がず・抜かず進める。

#### 主な術式
- **単束再建**：通常のACL再建。多くの施設で標準
- **二重束再建**：解剖学的により忠実、回復にやや時間
- **追加処置**：半月板修復／部分切除、軟骨修復、外側補強（ALL/LET）など

### 5-B. リハビリで重要な考え方

1. **時期＋基準のダブル達成**：時期だけ・基準だけ、どちらか一方では復帰しない
2. **移植腱の生物学的成熟**：再建腱は**6〜12か月**かけて生物学的に成熟する。早期復帰のリスクの根拠
3. **大腿四頭筋強度の早期改善**：膝の保護・再損傷予防の最重要因子
4. **動作の質を見る**：左右対称・膝外反コントロール・体幹姿勢
5. **心理的準備**：物理的に整っても心理が伴わない場合は復帰を急がない（ACL-RSI）

### 5-C. フェーズ別ゴール

> 期間・数値は目安。移植腱・付加術式により大きく変動。

#### フェーズ1：保護期（術後 0〜2週）
- **目標**：腫脹・疼痛管理、完全伸展（0°）獲得、四頭筋賦活、安全な荷重と移動
- **荷重**：医師指示（多くは耐性に応じた全荷重 / 装具併用）
- **ROM目安**：屈曲90°目安、**伸展は0°必須**
- **介入**
  - アイシング、患肢挙上、足関節ポンピング
  - 大腿四頭筋セッティング、SLR
  - 自動／他動ROM
  - 装具・松葉杖での歩行訓練
  - **NMES**（大腿四頭筋）

#### フェーズ2：可動域・筋力期（術後 2〜6週）
- **目標**：屈曲ROM拡大、四頭筋・ハム強化、歩行正常化
- **ROM目安**：屈曲120°前後、伸展0°維持
- **介入**
  - クォーター〜ハーフスクワット（疼痛なし）
  - レッグプレス（深さ制限）
  - エルゴメーター（サドル高めから）
  - クローズドキネティックチェーン優先
  - **オープンキネティックチェーンは医師許可下で慎重に**（特にBPTB／QT）
  - バランス訓練

#### フェーズ3：強化期（術後 6〜12週）
- **目標**：全可動域獲得、漸増抵抗運動、機能動作の対称性
- **介入**
  - フルスクワット（フォーム重視）
  - シングルレッグスクワット、ステップアップ／ダウン
  - エルゴメーター、ローイング
  - 階段昇降、ランジ系
  - 漸増抵抗運動（高負荷）

#### フェーズ4：ランニング・プライオ準備期（術後 12〜16週）
- **目標**：ランニング開始準備、ジャンプの基礎
- **進行基準（目安）**
  - 大腿四頭筋LSI≥80%
  - 疼痛・腫脹なし
  - シングルレッグスクワット良好
- **介入**
  - 直線走行（短距離→距離増加）
  - 両脚ジャンプ（ボックスジャンプ低）
  - ラテラル動作の基礎

#### フェーズ5：プライオメトリクス・アジリティ期（術後 16〜24週）
- **目標**：ジャンプ・着地の対称性、方向転換、スポーツ特異動作の段階的導入
- **進行基準（目安）**
  - LSI≥85%（筋力・ホップ）
  - フォーム良好
- **介入**
  - 片脚ジャンプ、ランディング訓練
  - カッティング、Tドリル
  - スポーツ特異練習（コンタクトなし）

#### フェーズ6：復帰準備期（術後 6〜9か月）
- **目標**：競技レベル動作への適応、再損傷リスク低減
- **介入**
  - チーム練習合流（コンタクトなし→徐々にあり）
  - スポーツ特異の高負荷トレーニング
  - 心理的準備の評価（ACL-RSI）

#### フェーズ7：競技復帰判定（術後 9か月以降）
- **復帰基準（目安／後述）を全てクリア**
- 医師の最終許可
- ACL-RSI 60〜65点以上目安
- **9か月以前の復帰は推奨しない**（S2, S4）

### 5-D. 復帰基準（一般的目安）

下記をすべて満たすことが望ましい（施設で具体的閾値は異なる）：

| 領域 | 基準 |
|------|------|
| ROM | 完全可動域、対側差なし |
| 大腿四頭筋筋力 | LSI≥90%（等速性測定） |
| ハムストリング筋力 | LSI≥90% |
| シングルホップ | LSI≥90% |
| トリプルホップ | LSI≥90% |
| クロスオーバーホップ | LSI≥90% |
| 6mタイマーホップ | LSI≥90% |
| 着地動作 | 対称、膝外反コントロール良好 |
| 心理 | ACL-RSI≥60〜65点 |
| 時期 | 術後9か月以降が望ましい |
| その他 | 疼痛・腫脹なし、コーチ・医師の合意 |

> **注**：LSI 90%は「健側比」。健側が術前から低下している場合は過大評価される可能性があるため、**術前値や標準値との比較**も併用。

### 5-E. 評価項目

| カテゴリ | 指標 |
|---------|-----|
| 疼痛 | NRS / VAS |
| 腫脹 | 周径測定 |
| ROM | 屈曲・伸展 |
| 筋力 | 等速性（HHD/Biodex）、MMT |
| ホップ | シングル・トリプル・クロスオーバー・6mタイマー |
| バランス | Y-Balance Test |
| PRO | KOOS、IKDC、Lysholm、Tegner |
| 心理 | ACL-RSI（Webster 2008、短縮版） |

詳細は `09_膝関節_評価項目一覧.md` 参照。

### 5-F. 介入例

#### 各フェーズの代表的種目（一部）

- **フェーズ1**：四頭筋セッティング、SLR、自動ROM、NMES
- **フェーズ2**：ハーフスクワット、レッグプレス、エルゴメーター、両脚バランス

### 5-H. 患者向け説明例（M1モード）

> 「ACL再建後のリハビリは**長期戦**です。痛みが治まるのは早くても、**靱帯が生物学的に成熟するには9〜12か月以上かかる**ため、見た目に元気でも内部はまだ回復途中、ということが起きます。
>
> 競技復帰の目安は**術後9か月以降＋筋力・ジャンプ・心理面の基準を満たす**ことです。早く戻りたい気持ちは当然ですが、**早期復帰は再断裂のリスクを大きく高める**ことが研究で示されています。
>
> 焦らず、リハビリの先生・主治医と一緒に基準を一つずつクリアしていきましょう。」

### 5-I. 医療者向け注意点

- **オープンキネティックチェーン（OKC）の取り扱い**：BPTB・QTは膝伸展機構への負荷を慎重に。レッグエクステンション（フル可動域）の早期導入は議論あり。
- **大腿四頭筋強度の改善が遅れがちなBPTB／QT症例**：NMES、BFR（血流制限訓練）の併用検討。
- **ハムストリング再建での持続的なハム弱化**：等速性で定期的に評価し、ハム特異強化を組み込む。
- **半月板修復併用例**：4〜6週の荷重・深屈曲制限が加わる。詳細は `06_半月板縫合術_術後リハ.md` 参照。
- **軟骨修復併用例**：6〜8週以上の保護期。荷重面か非荷重面かで配慮が異なる。
- **二次予防（ACL予防 神経筋トレーニング）**：再建後も**反対側ACL損傷リスク**は健常者より高い。FIFA 11+などの予防プログラムを継続。
- **女子・若年アスリート**：再損傷率が高く、予防プログラムの徹底が重要。
- **心理面のフォロー**：ACL-RSI低値は復帰後も再損傷リスクと関連。必要時に心理サポート紹介。
- **共有意思決定**：「6か月で復帰したい」と希望される場合のリスクコミュニケーション資料を準備。

### 5-J. 図解化する場合の見せ場

1. **フェーズ別タイムライン** — 7フェーズと主要な介入の時系列
2. **復帰基準のチェックリスト** — LSI 90%×4ホップ＋筋力＋心理＋時期
3. **移植腱別の特徴** — BPTB／HT／QTの3列比較

5. **着地時の膝外反** — 良いフォーム vs 悪いフォーム

## 6. 患者向けFAQ

#### Q1. 「いつスポーツに戻れますか？」
> 一般的に**術後9か月以降**が安全な目安と言われていますが、**筋力・ジャンプ・心理面の基準**を満たすことが最も大切です。早く戻ると**再断裂**のリスクが高まることが研究で示されています。

#### Q2. 「ランニングはいつから？」
> 多くの場合**術後12〜16週**から、まずは直線をゆっくり走ることから始めます。**痛み・腫れがない、太ももの筋力が健側の80%以上**などの条件を確認してから開始します。

#### Q3. 「ジャンプはいつから？」
> 一般的に**術後16〜20週以降**で、両脚ジャンプから始め、徐々に片脚ジャンプ・スポーツ特異動作に進みます。判断は担当者と確認しましょう。

#### Q4. 「採取した腱の種類で何が違いますか？」
> 主に3種類あります。
> - **膝蓋腱（BPTB）**：固定が強いが膝前面痛が出やすい、膝をついた姿勢が辛い時期がある
> - **ハムストリング腱（HT）**：採取部痛が少ないが、ハムストリングの筋力低下が残りやすい
> - **大腿四頭筋腱（QT）**：比較的新しい選択肢。腱採取部の保護が必要
>
> どの腱でも**復帰までの時間と基準は本質的に同じ**ですが、強化すべき筋肉のバランスが違います。

#### Q5. 「半月板も縫った場合は違いますか？」
> はい。**深く曲げる動作・荷重・ひねり動作**を術後4〜6週は制限することが多いです。担当医の指示を厳守してください。

#### Q6. 「再断裂が怖いです。」
> 不安なお気持ちは当然です。**復帰を急がず、基準を一つずつクリアする**ことが最大の予防です。心理的な不安が強い場合はスポーツ心理士のサポートを受けることも検討できます。

#### Q7. 「反対の膝も心配です。」
> 統計的に**反対側ACL損傷のリスクは健常者より高い**ことが報告されています。両側の予防プログラム（バランス・神経筋トレーニング）を継続することが推奨されます。

#### Q8. 「もし筋力が左右同じにならなかったら？」
> 強化を続ければ多くの方は健側比90%以上に達しますが、達しない場合は**復帰時期を延期**し、**強化期間を延ばす**ことが安全です。担当者と相談してプランを調整します。
>
> なお、筋力テストで90%以上になっても、それだけで復帰が決まるわけではありません。反対側の脚も弱っている場合があり、ジャンプ動作の質、不安感、医師・療法士の評価を合わせて総合的に判断します。

## 7. 医療者向け追加メモ

- **9か月ルールの意義**（Grindem 2016 / S4）：時期と基準達成を組み合わせると**再損傷リスクが84%減少**。単純化された判定ルールが臨床的に有用。
- **LSI 90%の落とし穴**：健側自体が脱コンディショニングしていると過大評価。**プレ受傷値・年齢・性別の標準値**との比較を併用。
- **着地動作の動画解析**：左右対称・膝外反・体幹コントロールを動画で評価する施設が増加。
- **BFR（血流制限訓練）**：早期の四頭筋強化に有効性が示されている（高負荷を避けつつ筋力増強）。
- **OKC vs CKC**：以前はOKC禁忌の論調があったが、現在は**段階的・部分可動域での組み込み**が許容される傾向。施設方針を確認。
- **再建腱の生物学的成熟**：6〜12か月かけて靱帯化（ligamentization）が進む。この過程で一時的に脆弱化する時期があると言われる。
- **女子アスリート対応**：女子は男子より再損傷率が高い。月経周期・ホルモン環境を含めた配慮が議論されている。
- **二次予防プログラム**：FIFA 11+、Sportsmetrics、PEPなど。継続性が成果に直結。
- **ACL-RSIの活用**：6か月時点のスコアが12か月時点の復帰可否を予測するエビデンスあり（Webster）。

## 8. 危険徴候・中止基準

- 膝の急な動揺感、ガクッと抜ける感じ（再断裂疑い）
- 急な強い疼痛、腫脹増悪
- 着地時の膝崩れ
- 創部の発赤・熱感・滲出（感染）
- 下腿の急な腫脹（DVT）
- 呼吸困難・胸痛（PE）
- 神経症状（しびれ・感覚低下の進行）
- 心理的不安が強く動作恐怖が強い場合（運動の質に影響）
- 詳細は `00_膝関節リハGPT_安全ルール.md` 5-D 参照
