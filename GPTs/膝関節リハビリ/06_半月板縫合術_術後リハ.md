## ⚠️ 最重要免責

- 半月板**縫合**（meniscal repair）は半月板**部分切除**（partial meniscectomy）とは**別物**。リハ進行は大きく異なるため混同しない。
- 縫合された組織は**生物学的治癒に時間がかかる**ため、**深屈曲・荷重・回旋動作の制限**を一定期間設ける。
- 損傷部位（外周・中央）、損傷形態（縦断・水平・複雑型・ラジアル・ルート）、修復技術によって制限内容が変わる。
- 本書の期間・荷重・角度は**目安**。医師指示・施設プロトコルが優先される。
- 復帰を急ぐと**再断裂・修復破綻**につながる。「治癒に時間をかける」のが鉄則。

## 3. 共通している内容

### 3-1. 損傷形態と部位で進行が大きく変わる
S1, S2, S3で共通。**安定型（縦断・ラムプ）と不安定型（水平・ラジアル・複雑型・ルート）でリハ進行差が顕著**。

### 3-2. 深屈曲は4か月程度避ける
S1で明示：**深いしゃがみ込み・ジャンプ・回旋動作は最低4か月避ける**。

### 3-3. 荷重制限は損傷型で異なる
| 損傷型 | 荷重 |
|--------|------|
| 縦断（vertical longitudinal）、ラムプ | 比較的早期から荷重可（術後早期） |
| 水平、ラジアル、複雑、ルート | **4〜6週の荷重制限**を設けることが多い |

### 3-4. 屈曲位での荷重を段階的に
- 術後**4週まで**：荷重位での膝屈曲は**45°以下**に制限
- 術後**8週まで**：荷重下の深屈曲（>90°）は避ける
- 術後**13〜16週**で60〜90°のミニスクワット段階的拡大

### 3-5. 復帰は最低4か月以降
S1, S2, S3共通：**完全な活動・スポーツ復帰は最低4か月以降**、基準達成が前提。

### 3-6. 関節線圧痛・腫脹は治癒不良サイン
S1：**joint line tenderness（関節線圧痛）または effusion（関節液貯留）**は修復破綻・治癒遅延のサイン。

## 5. GPT Knowledge本体

### 5-A. 概要

半月板縫合は損傷した半月板を**温存して縫い合わせる**術式。部分切除と異なり、**生物学的治癒**を待つ必要があるため、**長期的な保護**が前提。

#### 主な損傷形態と治癒の特徴
| 損傷型 | 治癒のしやすさ | リハ進行 |
|--------|----------------|----------|
| 縦断（vertical longitudinal）外周 | 良好（血流あり） | 比較的早期に進行可 |
| ラムプ（ramp） | 良好 | 早期可 |
| 水平（horizontal） | 中等度 | 4〜6週制限 |
| ラジアル（radial） | やや困難 | 4〜6週制限 |
| 複雑型 | 困難 | 6週以上 |
| ルート（root） | 特殊（伸展時の負荷大） | 6〜8週制限、深屈曲長期回避 |

### 5-B. リハビリで重要な考え方

1. **生物学的治癒を最優先**：3〜4か月は組織の治癒を守るのが最重要
2. **深屈曲・荷重屈曲・回旋を慎重に**：これらが修復部に最大ストレスをかける
3. **損傷型と部位で個別化**：プロトコルを画一化しない
4. **段階的進行**：時期と基準のダブルでクリア
5. **再損傷シグナルを見逃さない**：関節線圧痛・腫脹は治癒不良の警告

### 5-C. フェーズ別ゴール（一般的目安／単独半月板縫合）

> 損傷型・部位・修復技術で大きく変わる。医師指示優先。

#### フェーズ1：保護期（術後 0〜4週）
- **目標**：腫脹・疼痛管理、伸展ROM、四頭筋賦活、修復部保護
- **荷重**：医師指示。**安定型は早期荷重可、不安定型はPWB〜NWB**
- **ROM制限（損傷型による）**：
  - 安定型（縦断断裂・ラムプ病変など）：医師指示に従い、制限最小限〜制限なしの場合あり
  - 不安定型（水平断裂・ラジアル断裂・複雑断裂・ルート損傷など）：屈曲90°以下などの制限を設ける例が多い
  - ACL再建術などを併用している場合は、半月板処置・靱帯再建・軟骨損傷の中で最も保護が必要な条件を優先する
  - 必ず手術記録の損傷型・縫合方法・医師指示を確認する
- **荷重位での屈曲制限**：**45°以下**
- **介入**
  - アイシング、患肢挙上、足関節ポンピング
  - 大腿四頭筋セッティング、SLR
  - 膝伸展ROM（過伸展を避ける）
  - 装具・松葉杖併用での歩行
  - **避ける**：深屈曲・回旋・スクワット・しゃがみ・正座

#### フェーズ2：可動域・基本動作期（術後 4〜8週）
- **目標**：屈曲ROM拡大、歩行正常化、軽度筋力強化
- **荷重**：基本的に全荷重へ移行（医師許可下）
- **荷重位での屈曲制限**：**90°まで**
- **介入**
  - ROM拡大（ヒールスライド、座位での膝屈曲）
  - クォータースクワット（30°前後まで）
  - エルゴメーター（サドル高めで負荷軽い）
  - バランス訓練（両脚→片脚）
  - **避ける**：深いしゃがみ込み、ジャンプ、回旋動作

#### フェーズ3：強化期（術後 8〜13週）
- **目標**：筋力強化、機能動作の対称性
- **介入**
  - ハーフスクワット（45°程度）
  - レッグプレス（深さ制限あり）
  - シングルレッグスタンス、ランジ系（浅め）
  - エルゴメーター（負荷増、サドル下げる）
  - **避ける**：90°以上の負荷スクワット、ピボット動作、ジャンプ

#### フェーズ4：機能回復期（術後 13〜16週）
- **目標**：深屈曲方向への段階的拡大、ランニング準備
- **荷重位での屈曲制限**：**60〜90°まで段階的に拡大**
- **介入**
  - スクワット深さ拡大
  - 直線軽い走行（医師許可下）
  - スポーツ特異動作の準備（コンタクトなし、回旋なし）
  - **避ける**：強い回旋、ジャンプ・深いしゃがみは医師許可後

#### フェーズ5：復帰準備期（術後 4〜6か月）
- **目標**：スポーツ特異動作の段階的導入、復帰判定
- **介入**
  - ランニング、ジャンプ、カッティングの段階的導入
  - スポーツ特異練習
- **復帰判定**：基準達成（後述）と医師最終許可

### 5-D. 慎重に扱う動作・場面

| 動作 | 制限の目安 |
|------|-----------|
| **深屈曲（>120°）** | 最低4か月、ルート修復は6か月以上 |
| **正座** | 最低4〜6か月、その後も個別判断 |
| **しゃがみ込み（>90°負荷）** | 最低4か月 |
| **ジャンプ** | 最低4か月、基準達成後 |
| **回旋動作・カッティング** | 最低4か月、段階的に |
| **荷重下90°以上の屈曲** | 8週以降、段階的に |
| **荷重下45°以上の屈曲** | 4週以降、段階的に |

### 5-E. 復帰基準（一般的目安）

| 領域 | 基準 |
|------|------|
| 痛み | 安静時・運動時ともになし |
| 腫脹 | 残存しない |
| 関節線圧痛 | なし |
| ROM | ほぼ完全（健側比） |
| 筋力（四頭筋） | LSI≥90% |
| ホップテスト | LSI≥90%（4種すべて） |
| バランス | Y-balance、シングルホップ良好 |
| 動作の質 | 着地対称、膝外反コントロール |
| 時期 | 最低4か月、損傷型により6か月以上 |
| 医師許可 | 必須 |

### 5-I. 患者向け説明例（M1モード）

> 「縫った半月板は**しっかり治るのに3〜4か月以上かかります**。痛みが早く引いても、内部はまだ治癒の途中ということがよくあります。
>
> 特に注意したいのは、**深くしゃがむ・ジャンプする・ひねる動作**です。これらは縫い目に強い力をかけるため、**最低4か月**は避けたほうが安全です。
>
> 早く戻りたい気持ちはわかりますが、ここで焦ると**もう一度同じ手術が必要**になることもあります。担当者と一緒に、ゆっくり段階的に進めていきましょう。」

### 5-J. 医療者向け注意点

- **損傷型・部位の確認**：手術記録から損傷型、修復された部位、修復方法（縫合糸の数・パターン）を確認。施設プロトコルとの整合を取る。
- **ルート修復の特殊性**（S6）：ルート損傷は伸展時の負荷が大きく、伸展位の保護期間を長く取る場合がある。
- **修復部の血流**：外周は血流良好、中央部は血流乏しく治癒遅延しやすい。
- **ACL再建併用例**：荷重制限はACL単独より厳しい（4〜6週）が、最終的なリハは半月板に律速される。
- **若年・スポーツ選手の早期復帰圧**：エビデンスをもとにリスクコミュニケーションを徹底。
- **DREAM Trial（S5）の知見**：若年外傷性損傷でも**12週の運動療法が手術と同等の12か月成績**。手術一択ではない選択肢の存在を理解。
- **変性型半月板**（中高年）：手術より**運動療法が第一選択**（S4 Solsona-Carcas 2024 NMA）。
- **関節線圧痛・腫脹のモニタリング**：治癒不良の警告。再評価とMRI検討。

### 5-K. 図解化する場合の見せ場

1. **損傷型別の血流マップ** — 治癒しやすい部位／しにくい部位
2. **荷重位屈曲制限のタイムライン** — 4週・8週・13週の境界
3. **避けるべき動作カード** — 深屈曲・正座・ジャンプ・回旋を視覚化
4. **段階的スクワット深さ** — 30°→45°→60°→90°の進行
5. **復帰基準チェックリスト** — 9項目の達成度

## 6. 患者向けFAQ

#### Q1. 「いつから歩けますか？」
> 損傷の場所と縫い方によります。**安定型の縫合**なら早期から荷重OK、**複雑型・ラジアル・ルート**は4〜6週の荷重制限を設けることが多いです。**主治医の指示**を必ずご確認ください。

#### Q2. 「どうして深く曲げてはいけないの？」
> 膝を深く曲げると、**縫った糸に強い力**がかかるためです。最低4か月は治癒を優先して、深屈曲を避けます。早く曲げたい気持ちはわかりますが、ここでの我慢が将来を守ります。

#### Q3. 「正座はいつからできますか？」
> 一般的に**最低4〜6か月**は避けることが多いです。その後も**痛みなく安全にできるか**を確認しながら段階的に進めます。元の正座が完全にできるかどうかは個人差があります。

#### Q4. 「ジャンプ系のスポーツに戻れますか？」
> 多くの場合戻れますが、**最低4か月、基準を満たしてから**です。基準には筋力（健側比90%以上）・ホップテスト・着地動作の対称性などがあります。

#### Q5. 「ひねる動作はいつから？」
> 軽いひねりは段階的に取り入れますが、**スポーツでの強い回旋・カッティング**は**最低4か月**は避けます。

#### Q6. 「腫れが少し戻ってきました。」
> **関節液の再貯留**は治癒の警告サインの可能性があります。運動量を一旦下げ、**早めに担当者に相談**してください。

#### Q7. 「どのくらいで普通に生活できますか？」
> デスクワーク程度なら**2〜4週**で復帰する方も多いですが、**しゃがむ・ひねる動作**を含む生活（介護・農作業・大工仕事など）はもっと先になります。具体的な復帰時期は担当者と相談しましょう。

#### Q8. 「もし縫い目が外れたらどうなるんですか？」
> 半月板の修復破綻は**痛み・腫れ・引っかかり**として現れることが多いです。その場合は再手術や部分切除が選択肢になることがあります。**自己判断で激しい動きを試さない**ことが予防になります。

## 7. 医療者向け追加メモ

- **早期荷重（縦断・ラムプ）の根拠**：周辺血流が豊富で安定した縦断損傷では、**荷重ストレスが治癒を促す可能性**が示されている（S2, S3）。
- **ルート修復の伸展位保護**：歩行時の伸展で半月板が押し出される（extrusion）リスク。装具併用や歩行制限が選択肢。
- **荷重位屈曲制限の生体力学的根拠**：荷重下で膝屈曲角度が増えるほど半月板への負荷増加。45°→90°の段階制限はこれを反映。
- **深屈曲制限の解除**：医師許可・痛みなし・治癒所見良好を確認の上、段階的に。
- **MRIによる治癒確認**：6か月時点での再評価が選択肢。施設方針による。
- **二次的な変性予防**：半月板機能温存のメリットは長期的（変性関節症予防）。患者教育で長期視点を共有。
- **リハの監視下 vs 在宅**：難しい時期（4〜13週の段階制限）は**監視下リハが望ましい**。
- **疼痛・腫脹日記**：患者に毎日記録を促し、悪化の早期発見につなげる。

## 8. 危険徴候・中止基準

- 関節線圧痛の出現または増悪（治癒不良警告）
- 関節液貯留（腫脹）の再発・増悪
- 急な疼痛増悪
- 関節のロッキング、引っかかり（修復部破綻の疑い）
- 膝の動揺感、ガクッと抜ける感じ
- 創部の発赤・熱感（感染）
- 下腿の急な腫れ・痛み（DVT）
