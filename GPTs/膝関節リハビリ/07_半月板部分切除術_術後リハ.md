## ⚠️ 最重要免責

- 半月板**部分切除**（partial meniscectomy）は半月板**縫合**とは進行が異なる。**より早期に進めることが多い**が、油断は禁物。
- 切除量・部位（外周／中央）・関節軟骨の状態によって個別化が必要。
- 本書の期間・基準は**目安**であり、医師指示・施設プロトコルが優先される。
- 「早く戻れる」は事実だが、**腫脹・疼痛が残る場合は段階を急がない**。
- 中高年の変性型半月板に対しては**運動療法が第一選択**であり、手術は必須でないことを理解する（DREAM Trial、Solsona-Carcas NMA）。

## 3. 共通している内容

### 3-1. 縫合より早期に進む
S1, S2, S5-S7で共通：**部分切除は組織を修復しないため、生物学的治癒待ちが不要**。早期荷重・早期ROM拡大が可能。

### 3-2. 全荷重への移行が早い
- 術後初日〜数日で**痛みに応じた全荷重**（pain-limited weight bearing）
- 多くの場合、**1週間以内に松葉杖を離脱**

### 3-3. 標準プロトコルは8〜12週
- 平均8〜12週でリハビリ完了
- ランニングは術後**4〜6週**で開始されることが多い
- スポーツ復帰は**6〜8週から、最大3か月程度**

### 3-4. 復帰時期の幅が大きい（19〜89日）
S2レビュー：**19〜89日**で復帰する範囲。年齢・競技レベル・損傷型により大差。

### 3-5. 基準ベースの進行
S1で強調：**時期だけで決めず、機能基準を達成して進む**。
- 腫脹なし
- ROM完全
- 大腿四頭筋筋力対称
- 動作の質良好

### 3-6. 大腿四頭筋強度が復帰の最重要因子
S2：**Quad strength**が復帰可否を最も律速する。

- 手術 vs 運動療法：**運動療法が第一選択**（S3, S4）
- 手術後の効果は短期的に運動療法と同等以上だが、長期的には差が縮小

## 5. GPT Knowledge本体

### 5-A. 概要

半月板部分切除は、**損傷部の半月板組織を一部切除する**術式。縫合と異なり生物学的治癒を待つ必要がないため、**早期にリハビリが進められる**。

#### 主な適応
- 外傷性の半月板損傷で**修復困難な部位**（中央部の血流乏しい部位）
- 機械的症状（ロッキング、ひっかかり）が著しい
- 縫合に向かない損傷形態（広範な水平断裂など）
- 中高年の変性型半月板損傷（ただし**運動療法が第一選択**）

#### 縫合術との比較

| 項目 | 部分切除 | 縫合 |
|------|---------|------|
| 治癒待ち | 不要 | 3〜4か月以上 |
| 荷重 | 早期可 | 損傷型により制限 |
| 深屈曲 | 段階的に早期可能 | 4か月以上慎重 |
| 復帰 | 6〜8週〜3か月 | 最低4か月 |
| 長期予後 | 軟骨摩耗リスクあり | 半月板機能温存 |

### 5-B. リハビリで重要な考え方

1. **「早く進めること」と「無理しないこと」の両立**：早期進行は可能だが、**腫脹・疼痛が残る場合は段階を急がない**
2. **大腿四頭筋強度の早期回復**：復帰の最重要因子
3. **動作の質**：左右対称、膝外反コントロール
4. **長期視点**：部分切除は**長期的な軟骨摩耗・OA進行のリスク**があると言われる。生涯にわたる膝のケアが重要
5. **基準ベース**：時期だけで進めない

### 5-C. フェーズ別ゴール（一般的目安）

> 切除量・部位・軟骨状態で個別化。医師指示優先。

#### フェーズ1：急性期（術後 0〜1週）
- **目標**：疼痛・腫脹管理、歩行自立、ROM獲得開始
- **荷重**：痛みに応じた全荷重（医師指示優先）
- **ROM目安**：屈曲90°前後、伸展0°
- **介入**
  - アイシング、挙上、足関節ポンピング
  - 大腿四頭筋セッティング、SLR
  - 自動／自動介助ROM
  - 松葉杖から早期離脱（多くは1週以内）

#### フェーズ2：基本動作期（術後 1〜2週）
- **目標**：ROM拡大、歩行正常化、軽い筋力強化
- **ROM目安**：屈曲120°前後、伸展0°
- **介入**
  - ヒールスライド、座位膝屈曲
  - クォータースクワット
  - 両脚バランス
  - エルゴメーター（軽負荷）
  - 階段昇降練習（手すり使用）

#### フェーズ3：強化期（術後 2〜4週）
- **目標**：筋力強化、機能動作の対称性、軽い活動への復帰
- **介入**
  - ハーフスクワット、ランジ
  - レッグプレス、シングルレッグスタンス
  - エルゴメーター負荷増、屋外歩行
  - ステップアップ／ダウン

#### フェーズ4：復帰準備期（術後 4〜6週）
- **目標**：ランニング開始、スポーツ特異動作の準備
- **進行基準**
  - 腫脹なし
  - ROM完全
  - 大腿四頭筋LSI≥80%
- **介入**
  - 直線軽走行 → 距離増加
  - ジャンプの基礎（両脚）
  - 方向転換練習（緩やかな角度から）

#### フェーズ5：競技復帰判定（術後 6〜12週）
- **目標**：基準達成と競技復帰
- **復帰基準**（後述）達成と医師許可

### 5-D. 注意が必要な場面

- **腫脹・疼痛が残る場合**：段階を**戻して再評価**。「早く進める」を優先しない
- **軟骨損傷併用**：制限がやや厳しくなる
- **広範な切除**：将来的なOA進行リスクの説明
- **同時に骨棘除去・滑膜切除**：腫脹遷延が起きやすい

### 5-E. 復帰基準（一般的目安）

| 領域 | 基準 |
|------|------|
| 痛み | 安静時・運動時なし |
| 腫脹 | 残存しない |
| ROM | 完全（健側比） |
| 大腿四頭筋筋力 | LSI≥90% |
| ホップテスト | LSI≥90%（4種） |
| 動作の質 | 着地対称、膝外反コントロール |
| 時期 | 6〜8週以降が目安 |
| 医師許可 | 必須 |

### 5-F. 評価項目

| カテゴリ | 指標 |
|---------|-----|
| 疼痛 | NRS / VAS |
| 腫脹 | 周径測定 |
| ROM | 屈曲・伸展 |
| 筋力 | 四頭筋・ハム |
| ホップ | 4種ホップテスト |
| PRO | KOOS、IKDC、Lysholm |

詳細は `09_膝関節_評価項目一覧.md` 参照。

### 5-I. 患者向け説明例（M1モード）

> 「半月板の部分切除は、**痛みの原因になっていた部分だけを取り除く手術**です。縫合と違って組織の治癒待ちが不要なので、**比較的早く動ける**のが特徴です。
>
> ただし、**腫れや痛みがしばらく残る方**もいらっしゃいます。「早く戻れる」と聞いて焦って動くと、かえって長引くことがあります。**腫れていない・痛くない・力が入る**を確認しながら、段階的に進めていきましょう。
>
> また、長期的には**膝への負担を減らす生活習慣**が大切です。体重管理・適度な運動を一生続けることが、将来の膝を守ります。」

### 5-J. 医療者向け注意点

- **腫脹遷延例**：4週以降も腫脹が引かない場合は、活動量・運動内容を見直し、医師に再評価依頼。
- **切除量による予後**：広範切除ほど将来的なOA進行リスク。患者に長期視点を共有。
- **軟骨損傷併用**：chondroplasty・マイクロフラクチャー併用例は荷重制限が異なる。手術記録で確認。
- **変性型半月板**：手術前から既にOA変化があるケース多数。手術で症状改善するとは限らない。**手術前の保存療法選択**の意思決定支援が重要（S3, S4）。
- **大腿四頭筋強度の客観評価**：HHD・Biodexで定量化が望ましい。
- **再手術の判断**：症状再燃・新規ロッキングは再評価とMRI検討。
- **長期管理**：体重管理・有酸素運動の継続が将来のOA予防に重要。

### 5-K. 図解化する場合の見せ場

1. **縫合 vs 部分切除の比較** — タイムラインの違い
2. **段階的復帰タイムライン** — 0〜1週・2週・4週・6週
3. **大腿四頭筋強度回復のグラフ** — 健側比の推移
4. **長期視点でのケア** — 生涯にわたる膝の保護
5. **避けたい動作 vs 推奨動作** — 短期と長期の視点

## 6. 患者向けFAQ

#### Q1. 「いつから歩けますか？」
> 多くの場合、**手術翌日から痛みに応じて荷重**を始めます。松葉杖は1週間以内に外せる方が多いです。最終判断は主治医に。

#### Q2. 「いつから走れますか？」
> 一般的に**術後4〜6週**から、まずは直線をゆっくり走ることから始めます。**腫れがない・ROMが完全・筋力が回復**などの基準を満たしてから始めましょう。

#### Q3. 「いつスポーツに戻れますか？」
> 競技や個人差により**6週〜3か月**と幅があります。**基準を達成して医師の許可**が出てからの復帰が安全です。

#### Q4. 「縫合と何が違うの？」
> 縫合は**組織を縫って治癒を待つ**手術なので、最低4か月の保護が必要です。部分切除は**損傷部だけを取り除く**ので早く動けますが、**長期的に膝への負担**が残ることがあります。どちらが良いかは医師の判断によります。

#### Q5. 「腫れが引きません。」
> 個人差で**4週以降も軽い腫れが続く**ことがあります。ただし**急に腫れが増える・熱を持つ**などがあれば、すぐに担当者に相談してください。

#### Q6. 「将来、関節症になりますか？」
> 部分切除の長期データでは、**将来的にOAが進みやすい傾向**が報告されています。だからこそ、**体重管理・適度な運動・筋力維持**が大切です。怖がる必要はなく、生活習慣で予防できます。

#### Q7. 「正座はいつからできますか？」
> 個人差はありますが、**6〜8週以降に少しずつ試す**方が多いです。痛みのない範囲で、無理せず進めましょう。

#### Q8. 「中高年で半月板が悪いと言われたのですが、手術が必要ですか？」
> 中高年の変性型半月板の場合、**手術より運動療法が第一選択**とする研究が複数あります。手術で必ず痛みが消えるわけではないので、**まずは保存療法**を試すことが多いです。医師とよく相談してください。

## 7. 医療者向け追加メモ

- **DREAM Trial（S4）の知見**：若年外傷性損傷でも12週運動療法が手術と12か月成績で非劣性。即時手術の優位性が小さい。
- **Solsona-Carcas 2024 NMA（S3）**：変性型半月板病変ではPT介入が短期疼痛・伸展筋力で有意に優位。3か月以降は同等。
- **手術選択の意思決定支援**：症状・年齢・OA変化・活動レベルを統合した患者教育。
- **早期復帰の落とし穴**：腫脹・疼痛が残る場合に「早く動けるはず」と無理することで、長期化するケース。
- **長期OA進行**：部分切除後10〜20年で対側比較してOA進行率が高いとする報告がある。
- **体重管理の重要性**：BMI高値・若年で広範切除はリスク要因。長期教育を組み込む。
- **再手術リスク**：再損傷・新規損傷で再手術となるケースもあり、復帰後のフォロー継続が望ましい。
- **保険・社会復帰**：仕事復帰時期は職種で大きく異なる。事務職は1〜2週、肉体労働は4〜8週が目安。

## 8. 危険徴候・中止基準

- 創部の発赤・熱感・滲出（感染）
- 腫脹の急な増悪
- 関節のロッキング、引っかかりの再発
- 急な疼痛増悪
- 膝の動揺感
- 下腿の急な腫れ・痛み（DVT）
