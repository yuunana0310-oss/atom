> 膝関節リハビリ支援GPT用 Knowledgeファイル
> 用途：GPTの参照知識ベース。臨床判断・最終決定は医師指示および施設プロトコルに従うこと。

---

## ⚠️ 最重要免責

- 評価指標は**目的に応じて選択**するものであり、すべて測定する必要はない。
- 各指標の閾値（「LSI≥90%」など）は**研究の中央値・推奨**であり、施設・症例で調整される。
- 患者状態・併存疾患・年齢で適切な指標が異なる。
- 評価結果は**医療者の総合判断**の一部であり、単独で復帰可否を決めない。

## 3. 共通している内容

### 3-1. 指標は「目的別」に選ぶ
- **疾患特異PRO**：疾患・術式に応じて
- **動作能力**：機能評価
- **筋力・バランス**：客観評価
- **心理**：復帰判断時に

### 3-2. 信頼性・妥当性が確立された指標を優先
- KOOS, IKDC, Lysholm, Tegner はACL・OAで広く検証済み
- 多言語化対応のものが多い

### 3-3. PRO（患者立脚型）と客観評価の併用
- PROだけでは不十分（過大／過小評価）
- 客観評価だけでも不十分（生活の実感を反映しない）
- **両方を組み合わせる**ことが推奨

### 3-4. 経時的変化を見る
- 初診・退院・3か月・6か月・1年など、節目で再評価
- 単発の数値より変化の方向性が重要

## 4. 情報源ごとに違いがある内容

### 4-1. LSI ≥90% の絶対性
- 多くの研究・施設で復帰基準に採用
- 一方、健側がデコンディショニングしている場合**過大評価**となる懸念あり（S8関連の議論）
- 標準値・術前値との比較を併用する施設も増加

### 4-2. ACL-RSI 閾値
- 文献により**60点／65点／70点**などばらつき
- カットオフ値の標準化は議論中

### 4-3. KOOS と KOOS-JR の使い分け
- KOOS: 5サブスケール、汎用性高い
- KOOS-JR: 関節置換術特化、短縮版、APTA TKA CPG（S9）で推奨

## 5. GPT Knowledge本体

### 5-A. 概要

膝関節リハで使う評価指標は大きく以下に分類：

| カテゴリ | 役割 |
|---------|-----|
| 疼痛 | 主観的疼痛強度の数値化 |
| 関節可動域（ROM） | 動作能力の基礎 |
| 筋力 | リハの効果・復帰判定 |
| 動作能力（パフォーマンス） | 実生活・スポーツの基礎 |
| バランス・神経筋制御 | 再損傷予防・転倒予防 |
| ホップテスト | スポーツ復帰判定 |
| PRO（患者立脚型） | 患者の主観・生活実感 |
| 心理（ACL-RSI） | 復帰準備度 |
| 動作の質 | 動画解析・観察 |

### 5-B. 各指標の詳細

#### 1. 疼痛

| 指標 | 用途 | 範囲 | 出典 |
|------|------|------|------|
| **NRS（Numerical Rating Scale）** | 急性／慢性疼痛 | 0〜10 | 一般的 |
| **VAS（Visual Analog Scale）** | 同上 | 100mmスケール | 一般的 |

**MCID（最小臨床的有意差）の目安**：NRS で**2点改善**が臨床的に有意とされることが多い。

#### 2. 関節可動域（ROM）

| 指標 | 用途 |
|------|------|
| 屈曲角度 | ゴニオメーター・スマホアプリ |
| 伸展角度 | 完全伸展（0°）獲得が歩行の鍵 |
| 過伸展（hyperextension） | 5°〜10°の生理的範囲 |

**測定の標準化**：仰臥位／腹臥位、関節中心の触知、対側との比較。

#### 3. 筋力

| 指標 | 用途 | 注意 |
|------|------|------|
| **MMT（徒手筋力検査）** | 簡便、5段階 | 上限効果あり、4-5の判別困難 |
| **HHD（Hand-held Dynamometer）** | 定量、外来でも使用可 | 検者間信頼性に注意 |
| **等速性筋力（Biodex等）** | ゴールドスタンダード | 設備必要 |
| **30秒立ち上がりテスト** | 簡便、APTA TKA CPGで推奨 | 高齢者・OAで有用 |
| **片脚スクワット** | 機能評価 | 観察評価併用 |

**LSI（Limb Symmetry Index）= 患側 / 健側 × 100%**
- 一般的閾値：**90%**（復帰判定）
- 健側のデコンディショニングに注意

#### 4. 動作能力（パフォーマンス）

| 指標 | 用途 | 出典 |
|------|------|------|
| **TUG（Timed Up and Go）** | 動作能力・転倒リスク | APTA TKA CPG |
| **30秒立ち上がり（30 sec STS）** | 下肢筋力・耐久 | APTA TKA CPG |
| **歩行速度（10m歩行）** | 機能の総合指標 | 一般的 |
| **6分間歩行テスト（6MWT）** | 持久性 | 心肺機能評価兼用 |
| **階段昇降テスト** | 実生活動作 | 一般的 |

#### 5. バランス・神経筋制御

| 指標 | 用途 | 出典 |
|------|------|------|
| **片脚立ち時間** | 簡便、転倒リスク | 一般的 |
| **Y-Balance Test (YBT-LQ)** | 動的バランス、再損傷予測 | S4, S7 |
| **Star Excursion Balance Test (SEBT)** | YBTの原型 | S4 |
| **BESS（Balance Error Scoring System）** | 静的バランス、スポーツ復帰 | 一般的 |

**Y-Balance参考値**（S5, S7）：
- 異性差あり、男性が女性より長距離
- **anterior reach 左右差 >2 cm = 下肢損傷リスク上昇**
- 復帰判定：**LSI≥90%**を目安とすることが多い

#### 6. ホップテスト

| 指標 | 用途 |
|------|------|
| **シングルホップ（Single hop for distance）** | 片脚最大ホップ距離 |
| **トリプルホップ（Triple hop for distance）** | 連続3回ホップの合計距離 |
| **クロスオーバーホップ（Crossover hop）** | 一直線をジグザグでホップ |
| **6mタイマーホップ（6m timed hop）** | 6m片脚ホップでのタイム |

**ホップ判定**（S6）：
- **すべてのホップで LSI≥90%**が一般的閾値
- ACL再建後の復帰判定では「4種ホップ + 筋力」をパッケージ評価
- **着地動作の質**（フォーム）も併せて評価

#### 7. PRO（患者立脚型アウトカム）

| 指標 | 用途 | 範囲 | 特徴 | 出典 |
|------|------|------|------|------|
| **KOOS** | 膝損傷・OA、5サブスケール（疼痛・症状・ADL・スポーツ／レクリエーション・QOL） | 各0〜100、100=最高 | 汎用性高い、無料、多言語 | S1, S2 |
| **KOOS-JR** | 関節置換術特化（短縮版） | 0〜100 | APTA TKA CPG推奨 | S9, S10 |
| **IKDC**（IKDC Subjective Knee Form） | 膝靱帯損傷、術後評価 | 0〜100 | スポーツ選手向け | S1 |
| **Lysholm Knee Score** | 膝靱帯・半月板損傷、症状評価 | 0〜100 | 古典的、ACL研究で多用 | S1 |
| **Tegner Activity Scale** | 活動レベル | 0〜10 | 受傷前と比較で活動低下を評価 | S1 |
| **WOMAC** | 膝・股OA | サブスケール集計 | OAで広く使用 | S1 |
| **Oxford Knee Score (OKS)** | TKA・UKA特化 | 0〜48 | 関節置換のRCTで多用 | S1 |
| **Knee Outcome Survey - ADL Scale (KOS-ADL)** | 一般的膝損傷 | 0〜100 | 簡便 | S1 |
| **Activity Rating Scale (ARS)** | 活動レベル | 0〜16 | スポーツ別 | S1 |

**PRO測定の頻度**：初診、退院、3か月、6か月、1年が一般的。

#### 8. 心理（復帰準備度）

| 指標 | 用途 | 範囲 | 出典 |
|------|------|------|------|
| **ACL-RSI（Anterior Cruciate Ligament Return to Sport after Injury）** | ACL再建後の心理的復帰準備 | 12項目、0〜100点 | S3 |
| **ACL-RSI 短縮版（Webster 2018）** | 簡便版 | 6項目 | S3 |
| **TSK（Tampa Scale for Kinesiophobia）** | 運動恐怖 | 17/11項目 | 補助的 |
| **PSE（Pain Self-Efficacy Questionnaire）** | 自己効力感 | 0〜60 | 慢性痛 |

**ACL-RSI閾値の目安**：
- 文献により**60〜65点**が復帰準備の目安
- 6か月時点のスコアが12か月時点の復帰可否を予測（Webster）

#### 9. 動作の質（観察・動画解析）

| 指標 | 用途 |
|------|------|
| **着地時の膝外反（Dynamic knee valgus）** | ジャンプ着地・スクワットで評価 |
| **対称性** | 左右脚の動作パターン |
| **体幹コントロール** | 着地時の体幹側屈・前屈 |
| **LESS（Landing Error Scoring System）** | 着地動作のスコアリング |
| **TJA（Tuck Jump Assessment）** | プライオメトリクス動作評価 |

### 5-C. 疾患・術式別の推奨評価セット（一般的目安）

#### 膝OA保存療法
- 疼痛（NRS）、ROM、四頭筋筋力、TUG、30秒立ち上がり、KOOS／WOMAC

#### TKA / UKA術後
- 疼痛、ROM、四頭筋筋力（30秒立ち上がり）、TUG、KOOS-JR／OKS（APTA推奨）

#### ACL再建術
- 疼痛、ROM、四頭筋筋力（HHD/Biodex）、ハム筋力、ホップ4種、Y-Balance、KOOS、IKDC、Lysholm、Tegner、ACL-RSI、動作の質

#### 半月板縫合・部分切除
- 疼痛、関節線圧痛、腫脹、ROM、四頭筋筋力、ホップ、KOOS、IKDC

#### HTO
- 疼痛、ROM、四頭筋筋力、TUG、6分間歩行、KOOS、Tegner、X線（アライメント・骨癒合）

### 5-D. 評価頻度の目安

| タイミング | 内容 |
|-----------|-----|
| 初診 | フル評価（ベースライン） |
| 2週ごと（リハ期間中） | 疼痛、ROM、簡便な機能指標 |
| 4〜6週 | 中間評価、PRO測定 |
| 退院／プログラム終了時 | フル評価 |
| 3か月、6か月、1年 | 長期フォロー、PRO |

### 5-E. 介入例（評価結果に応じた）

- **疼痛高値・腫脹あり**：管理優先（アイス・挙上・運動量調整）
- **ROM不足**：ROM拡大エクササイズ重点
- **四頭筋筋力低下**：高強度筋力訓練、NMES、BFR
- **バランス低下**：Y-Balance改善目的の動的バランス訓練
- **ホップLSI低値**：プライオメトリクス、片脚機能訓練
- **ACL-RSI低値**：心理サポート、段階的暴露、コミュニケーション

### 5-F. 図解化する場合の見せ場

1. **評価カテゴリのマップ** — 9カテゴリの全体像
2. **疾患別推奨セット** — 表形式で一覧
3. **LSI 90% の意味** — 患側 / 健側のイメージ
4. **ホップテスト4種** — 動画／イラスト
5. **Y-Balance 3方向** — anterior / posteromedial / posterolateral

## 6. 患者向けFAQ

#### Q1. 「KOOSって何ですか？」
> 膝のさまざまな状態を**患者さん自身の感覚**で評価する質問紙です。痛み・症状・日常生活・スポーツ・QOLの5項目を答えると、**0〜100点**で結果が出ます。100点が最も良い状態です。

#### Q2. 「LSI 90%ってどういう意味？」
> LSI（Limb Symmetry Index）は**怪我した方の脚 ÷ 健康な方の脚**の比率です。**90%以上**を目安とすることが多く、左右差が少ないことを示します。

#### Q3. 「ホップテストはどうやるの？」
> 片脚で**できるだけ遠くに跳ぶ**テストや、**3回連続で跳ぶ**テスト、**6mを片脚で跳んでタイムを測る**などの種類があります。左右両脚で行い、対称性を評価します。

#### Q4. 「ACL-RSIって何を測るの？」
> ACL再建後の**心の準備状態**を測る質問紙です。スポーツに戻る不安・恐怖・自信などを点数化します。**身体だけでなく心の準備も大切**と考えられています。

#### Q5. 「測定はリハビリのたびに必要？」
> 毎回フル評価する必要はありません。**節目（2週・4〜6週・退院時・3か月・6か月・1年）**で重点的に評価し、間は簡便な指標（疼痛・ROM）でフォローします。

#### Q6. 「数値が良くないと復帰できないの？」
> 数値は**判断材料の一つ**です。基準を満たさない場合は、**強化期間を延ばす**ことが多いです。焦って復帰するより、しっかり準備するほうが再損傷を防げます。

## 7. 医療者向け追加メモ

### 7-A. 指標選択の原則
- **施設の標準セット**を決めておく（毎回違う指標を使わない）
- **無料で使える指標**を優先（KOOS、Lysholm、IKDC、Tegnerは無料）
- **多言語版**があるものを優先（国際比較）
- **MCID（最小臨床的有意差）**を把握しておく

### 7-B. 客観評価のセットアップ
- HHD（手持ち徒手筋力計）：外来環境で実用的
- Y-Balance Kit：簡便な動的バランス評価
- ストップウォッチ + メジャー：ホップテスト、TUG、6MWT
- 動画撮影：動作の質評価（同意取得後）

### 7-C. PRO収集の運用
- 紙ベース・電子化（QRコードでGoogleフォーム等）
- 来院時または来院前に自己回答
- 経時的変化をグラフ化して患者と共有（モチベーション向上）

### 7-D. 評価結果の記録
- カルテに数値を残す（経時比較のため）
- 患者にフィードバック（自己効力感向上）
- 進行判断の根拠として活用

### 7-E. 評価の落とし穴
- **LSIの過大評価**：健側コンディショニング低下
- **動作の質の見落とし**：数値クリアでもフォーム不良で再損傷
- **PROのみで判断**：客観評価との乖離
- **単発測定**：経時的変化を見逃す
- **疼痛の影響**：疼痛強い時の筋力測定は過小評価

### 7-F. 関連する研究知見
- **Grindem 2016（S8）**：時期＋基準達成で再損傷リスク84%減
- **Webster ACL-RSI**：6か月時点スコアの予測力
- **APTA TKA CPG（S9）**：KOOS-JR・30秒立ち上がり・TUGをBest Practiceで推奨

## 8. 危険徴候・中止基準

評価中に以下を観察した場合は中止し、医師に相談する：

- 急な強い疼痛、腫脹増悪
- 関節のロッキング、引っかかり
- 膝の動揺感、ガクッと抜ける
- 失神、立ちくらみ、強い動悸
- 評価動作で再損傷の懸念があるサイン
- 心理的に追い詰められた様子

詳細は `00_膝関節リハGPT_安全ルール.md` 5-D 参照。
