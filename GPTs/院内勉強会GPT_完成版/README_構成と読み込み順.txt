# 院内勉強会GPT 完成版 — 構成と読み込み順

このフォルダは、複数の旧ナレッジに存在した重複・矛盾を解消し、
機能を落とさずにエラー（同じ画像のループ・ルールの二重適用）を防ぐために整理した完成版一式です。

→ 00 は Instructions 欄へ。01〜12 は Knowledge（ファイル添付）へアップロードします。

## 旧版から修正・解消した点
1. 【重複解消】共通デザインディレクションは新版（枚数モード判定あり）を採用。
   旧版（枚数モード判定なし）は同梱しない。
   → 複数枚指示なのに1枚へコラージュ化＝似た画像の量産を防止。

2. 【重複解消】根拠確認ルールは改正版（Canva非依存に合わせた版）を採用。
   さらに改正版で抜けていた「Step5｜画像生成」を復活させ、Step番号を正常化（Step1〜6）。

3. 【記号衝突の解消】Instructions本体で、デザイン方向性をプリセット名称で提示するよう統一。
   テイスト記号A〜Lの定義は「02_デザインプリセットナレッジ」だけを基準とする。
   （以前は §5/§12/映画風制限/参照モード で A・B・C が別々の意味を持ち混乱していた）

4. 【生成ループ防止】Instructions §17 に「完了条件」を追加。
   生成後チェックを通れば完成として提示し、指示があるまで再生成しない。
   §18 の再生成は原則1回とし、改善しない場合は再生成を繰り返さずユーザーへ報告。

5. 【誤解防止】映画ナレッジ冒頭に注記を追加。
   「映画の1場面」は構図手法であり、シネマティック・テイストの自動選択ではない。

6. 【用途分離】患者・家族向けタイプ一覧の冒頭に注記を追加。
   医療者向け資料と1枚に混在させない／対象モードを先に判定する。

## 補足
・「光・陰影ナレッジ」(12) を追加済み。一部ナレッジ（01・02・03）からの参照先がこれで揃いました。
　ページ役割ごと（表紙＝強め／本文＝抑える／チェックリスト＝可読性優先）の光量配分は、
　共通デザイン(03)・プリセット(02)のページ役割別選択と整合しています。
