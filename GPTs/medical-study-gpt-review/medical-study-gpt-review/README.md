# medical-study-gpt-review

## 使い方

1. このフォルダをCodexに渡す
2. `knowledge/` フォルダに、今あるナレッジをそのまま追加する
3. `codex_first_prompt.md` の内容をCodexに投げる
4. まず監査レポートだけ作らせる
5. 監査結果を確認してから、改訂版の作成に進む

## 最初にやること

- `main_instruction.md`：現在の勉強会GPT本体指示
- `AGENTS.md`：Codexに守らせる作業ルール
- `knowledge/`：既存ナレッジを入れる場所
- `codex_first_prompt.md`：Codexへの最初の依頼文
