## 常勤と非常勤の定義

### 誤解されやすい点

多くの人は「常勤 = 正社員」「非常勤 = パート」と思っていますが、**法律上は異なります**。

---

### ✅ 正しい定義

```
【常勤の定義】

「施設が就業規則で定めた常勤職員の勤務時間に達している職員」

【非常勤の定義】

「施設が定めた常勤時間に満たない職員」
```

**つまり、重要なのは：**
- 雇用形態（正社員か、パートか）ではない
- **施設が定めた「常勤時間数」に達しているかどうか**

---

## 施設が定めた「常勤時間数」の重要性

### 常勤時間数の決定方法

施設が就業規則で「常勤職員の標準勤務時間」を定めます。

#### パターン1：週40時間を常勤とする場合

```
就業規則の定め：「常勤職員は週40時間勤務」

【正社員で週40時間】
→ 常勤 ✅

【パートで週35時間】
→ 非常勤 ✅

【パートで週40時間以上】
→ 常勤 ✅
  （雇用形態はパートでも、
   勤務時間が「常勤の基準」に達しているため）
```

#### パターン2：週32時間を常勤とする場合

```
就業規則の定め：「常勤職員は週32時間勤務」

【パートで週32時間以上】
→ 常勤 ✅

【パートで週30時間】
→ 非常勤 ❌
  （施設の常勤基準に満たないため）
```

## あなたの勤務形態の確認

### 情報を整理する必要があります

あなたは2つの情報を述べられています：

#### 情報1：「月～土で週30時間勤務」（最初）
```
月～土で週30時間
→ 平均すると1日約5時間？

この場合：週30時間 < 週40時間（標準）
→ 一般的には「非常勤」と判定される可能性が高い
```

#### 情報2：「パートでも日6時間勤務」（今）
```
月～土で毎日6時間
→ 月～土の営業時間によって異なる

例：営業が月～金のみで土は半日の場合
  月～金：6時間×5日 = 30時間
  土：6時間×0.5日 = 3時間
  合計：約33時間/週

例：営業が月～土フル営業の場合
  月～土：6時間×6日 = 36時間/週
```

## ⚠️ 正確な勤務時間数の確認が必須

```
Q: あん摩マッサージ指圧師の実際の勤務時間は？

A: 以下を確認してください：

  （週40時間か、週32時間か、その他か）

☐ あん摩マッサージ指圧師の契約内容：
  ・正社員か、パートか
  ・週何時間の勤務か（月～土の営業日数で計算）
  ・実際に月～土毎日6時間なら、週計何時間か

☐ パート契約でも、当院の「常勤基準」に達しているか
```

## パターン別の法的評価

### 【パターンA】週30時間以下のパートの場合

```
勤務形態：パート、週30時間以下
例：月～土で実質週30時間

法的評価：「非常勤」
```

#### 特養での配置の可否

```
加算を算定していない場合：
→ パートタイム配置 ✅ 可能

加算を算定している場合：
→ パートタイム配置 ❌ 不可
   理由：「常勤」要件を満たさないため
```

**リスク**: 高リスク（グレーゾーン）

---

### 【パターンB】週32時間以上40時間未満のパートの場合

```
勤務形態：パート、週32～40時間
例：月～土で実質週36時間

法的評価：
  施設の常勤定義による
```

#### 施設の常勤定義が「週32時間以上」の場合

```
→ 「常勤」として認定される可能性がある ✅

この場合、特養で：
  加算を算定している場合でも
  パートでも「常勤」と認定される可能性あり

リスク：中程度
（施設の就業規則の定義に依存）
```

#### 施設の常勤定義が「週40時間」の場合

```
→ 「非常勤」と判定される

リスク：高い（パート配置は加算要件違反の可能性）
```

---

### 【パターンC】週40時間以上のパートの場合

```
勤務形態：パート、週40時間以上
例：月～土で毎日6.67時間以上（実質週40時間以上）

法的評価：「常勤」 ✅
```

#### 法的根拠

```
出典：freee クラウド会計
[常勤換算とは？人員配置基準などの確認に必要な指標の計算方法]

「常勤とは、事業所が定めた規の勤務時間内で勤務している職員。
 雇用形態（正規・非正規）は関係ない。」

「パートタイマーでも、施設の定めた常勤時間数に達していれば
 常勤として認められる。」
```

#### 特養での配置の可否

```
加算を算定している場合：
→ 理論上、加算要件「常勤」を満たす ✅

複数人常勤換算配置シナリオ：
→ 「常勤換算」計上が正当化される可能性あり

リスク：低い（法的根拠が強い）
```

---

### 【パターンD】育児・介護・治療のための短時間勤務制度を利用している場合

```
勤務形態：パート、週30時間以上
例：育児のため週30時間短時間勤務

法的根拠：令和6年度介護報酬改定（2024年4月以降）

法的評価：「常勤」として扱うことが認められている ✅
```

#### 条件

```
以下のいずれかに基づく短時間勤務制度を利用している場合：

✅ 育児休業、介護休業等育児又は家族介護を行う労働者の
   福祉に関する法律に基づく短時間勤務制度

✅ 男女雇用機会均等法に基づく母性健康管理措置

✅ 厚生労働省「事業場における治療と仕事の両立支援のための
   ガイドライン」に沿った所定労働時間の短縮措置

「この場合、週30時間以上の勤務で常勤として扱える」
```

**出典**：  
[厚生労働省老健局 人員配置基準等 - 社会保障審議会資料](https://www.mhlw.go.jp/content/12300000/001144339.pdf)

#### 特養での配置の可否

```
加算を算定している場合：
→ 週30時間のパートでも「常勤」と認定される ✅

複数人常勤換算配置シナリオ：
→ 「常勤」配置として正当化される可能性あり

リスク：低い（法令で認められている）
```

| **A** | 週30時間以下 | 非常勤 | ❌ 不可 | 🔴 高い |

## 結論：パートでも「常勤」と認定される可能性

### ✅ パートタイムでも「常勤」と扱われる場合

```
1. 施設の就業規則で「常勤時間数」を週32時間以上と定めている
   + あん摩マッサージ指圧師がその時間数に達している

2. または、週40時間以上勤務している

3. または、育児・介護・治療のための短時間勤務制度を
   利用していて、週30時間以上勤務している
```

**この場合：**
- パートでも「常勤」と法的に認定される
- 加算の「常勤要件」を満たす可能性がある
- 複数人常勤換算配置のシナリオが成立する可能性がある

## 加算要件への影響

### 【重要な転換】

もし、あん摩マッサージ指圧師が「パートでも施設の常勤基準を満たしている」なら：

```
従来の評価：「パート = 非常勤 → 加算NG」
         ↓
新しい評価：「パートでも常勤基準達成 → 加算OK」
```

---

### シナリオの再評価

#### 複数人常勤換算配置（あん摩マッサージ指圧師パート + 看護職員兼務）

```
従来の評価：
  パート + 常勤換算 = グレーゾーン、リスク60-70%
  ↓
  推奨：都道府県事前相談

新しい評価（あん摩マッサージ指圧師が施設の常勤基準達成の場合）：
  「パート」でも「施設の常勤」 = 法的根拠が強化
  ↓
  リスク：30-40%に低減？
  推奨：それでも都道府県事前相談（慎重に）
```

## 次のステップ

### Step 1：正確な勤務時間数を確認

```
「月～土で毎日6時間」
→ 月～土が営業日なら、週36時間

「月～土で週30時間」
→ 不明（複数の解釈可能）

正確な契約書を確認してください
```

### Step 2：当院の就業規則を確認

```
「常勤職員の勤務時間数は？」
→ これにより、あなたが「常勤」か「非常勤」か判定される
```

### Step 3：特養と協議

```
「あん摩マッサージ指圧師が『施設の常勤基準』を
 満たしている場合、加算要件について
 どう判断されるか」

→ この点で特養の見解を確認
```

### Step 4：都道府県に相談（推奨）

```
正確な情報を揃えた上で、都道府県に相談
→ 「このような配置で加算算定は可能か」
```
