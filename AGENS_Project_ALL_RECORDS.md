# 💠 AGENS Project Strategy & Dev History (Memory)

本ファイルは、これまでの開発経緯、決定事項、技術仕様を保存するための「プロジェクト記憶」です。
AIが交代した場合や、将来プロジェクトを拡張する際に、このファイルを最初に読み込ませてください。

## 🎯 プロジェクトの共通目的
- **ゴール**: 士業（税理士・社労士等）や経営者をターゲットとした、完全ローカル稼働の「超自律型AI受託パッケージ」の製品化とBrainでの販売。
- **ターゲット市場**: 専門知識が武器になり、かつ機密保持（オフライン性）へのニーズが高い高単価マーケット。

## 💼 ビジネス戦略
- **販売プラットフォーム**: Brain（投資マインドの高い層が多い、アフィリエイト機能が強力なため）。
- **差別化要因**: 
  - 単なるプロンプト集ではなく、「5,700件の最新法律・実務Q&Aを搭載したAIチーム（4名）」であること。
  - プログラミング知識ゼロでも使える「専用ダッシュボード」と「ワンクリック起動」を完備。
  - セキュリティ：API利用でも学習に使われない、専門業種向けの信頼性。

## 👥 AI社員チーム（エージェント定義）
1. **Researcher (リサーチ担当)**: 官公庁公式ニュース・法改正の自動監視・要約。
2. **Expert Assistant (実務アシスタント)**: 5,700件の条文に基づく高度な判断・書類草案。
3. **Communicator (顧客窓口)**: 専門用語の「安心翻訳」、クライアントメール作成。
4. **SNS Strategist (SNS戦略案)**: 蓄積されたナレッジをバズるコンテンツへ変換。

## 🛠️ 技術スタック & 実装済み仕様
- **RAGエンジン**: ChromaDB (MiniLM-L12-v2 多言語モデル)
- **データ収集**: 
  - e-Gov API V2 連携（法律・施行令・施行規則の全文取得ロジック構築済み）
  - 国税庁・厚労省スクレイピング（180件以上の実務Q&A）
- **UI/UX**:
